## 一、量纲分析的基本思想

### 1.1 物理量的量纲

任何物理量都可以用**基本量纲**的幂次乘积来表示。

在国际单位制（SI）中，有 7 个基本量纲：

| 基本量 | 量纲符号 |
|:---:|:---:|
| 长度 | $L$ |
| 质量 | $M$ |
| 时间 | $T$ |
| 电流 | $I$ |
| 温度 | $\Theta$ |
| 物质的量 | $N$ |
| 发光强度 | $J$ |

> 在力学问题中，通常只需要 $M, L, T$ 三个基本量纲。

### 1.2 核心原则

> **任何正确的物理方程，其等号两边的量纲必须相同。**

更进一步：**任何有物理意义的函数关系，其自变量和因变量必须能组合成无量纲量。**

---

## 二、Buckingham $\pi$ 定理

### 2.1 定理陈述

设一个物理问题涉及 $n$ 个物理量 $q_1, q_2, \ldots, q_n$，这些量依赖于 $k$ 个**基本量纲**（即量纲矩阵的秩为 $k$），则：

$$\boxed{F(q_1, q_2, \ldots, q_n) = 0 \quad \Longleftrightarrow \quad f(\pi_1, \pi_2, \ldots, \pi_{n-k}) = 0}$$

其中 $\pi_1, \pi_2, \ldots, \pi_{n-k}$ 是 $n - k$ 个**独立的无量纲乘积**（$\pi$ 项）。

### 2.2 定理的含义

| 信息        | 说明                          |
| :-------- | :-------------------------- |
| **原来**    | 需要在 $n$ 维参数空间中寻找函数关系        |
| **现在**    | 只需在 $(n-k)$ 维无量纲参数空间中寻找函数关系 |
| **减少的维度** | $k$ 个（等于基本量纲的个数）            |

## 三、$\pi$ 定理的实施步骤

### 步骤流程图

```
列出所有相关物理量 (n个)
        ↓
确定基本量纲 (k个)
        ↓
选取 k 个"重复量"（每个基本量纲至少出现一次）
        ↓
构造 (n-k) 个无量纲 π 项
        ↓
写出无量纲函数关系 f(π₁, π₂, ...) = 0
        ↓
通过实验或理论确定函数 f 的具体形式
```

### 详细步骤

**Step 1：列出所有相关物理量**

写出影响该物理问题的所有变量：

$$q_1, q_2, \ldots, q_n$$

> **关键**：选多了浪费计算，选少了结论错误。需要物理直觉。

**Step 2：确定基本量纲的个数 $k$**

写出每个量的量纲，构建**量纲矩阵**，其秩即为 $k$。

**Step 3：选取 $k$ 个"重复量"（repeating variables）**

选取原则：
- 这 $k$ 个量的量纲必须**线性无关**（覆盖所有 $k$ 个基本量纲）
- 不能选取待求的因变量
- 优先选取几何量（$L$）、运动学量（$v, a$）、动力学量（$\rho, F$）

**Step 4：构造 $\pi$ 项**

对每个非重复量 $q_i$，令：

$$\pi_i = q_i \cdot q_{r_1}^{a_1} \cdot q_{r_2}^{a_2} \cdots q_{r_k}^{a_k}$$

令 $[\pi_i] = M^0 L^0 T^0$，解出指数 $a_1, a_2, \ldots, a_k$。

**Step 5：写出最终关系**

$$f(\pi_1, \pi_2, \ldots, \pi_{n-k}) = 0$$

或等价地将因变量的 $\pi$ 项写为其他 $\pi$ 项的函数。

## 四、经典例题

### 例 1：单摆周期

**问题**：单摆的周期 $T$ 与哪些因素有关？

**Step 1**：列出相关物理量

|   量   |    符号    |         量纲         |
| :---: | :------: | :----------------: |
|  周期   |   $T$    |       $T^1$        |
|  摆长   |   $l$    |       $L^1$        |
| 摆球质量  |   $m$    |       $M^1$        |
| 重力加速度 |   $g$    |    $L^1 T^{-2}$    |
|  摆角   | $\theta$ | $M^0 L^0 T^0$（无量纲） |

共 $n = 5$ 个量。

**Step 2**：基本量纲 $k = 3$（$M, L, T$）

**Step 3**：选取重复量 $l, g, m$（覆盖 $L, T, M$）

**Step 4**：构造 $\pi$ 项

**$\pi_1$（含 $\theta$）**：

$$\pi_1 = \theta \cdot l^{a_1} g^{b_1} m^{c_1}$$

$$[\pi_1] = M^0 L^0 T^0 \cdot L^{a_1} \cdot (L T^{-2})^{b_1} \cdot M^{c_1} = M^{c_1} L^{a_1+b_1} T^{-2b_1}$$

令各指数为零：
- $c_1 = 0$
- $-2b_1 = 0 \Rightarrow b_1 = 0$
- $a_1 + b_1 = 0 \Rightarrow a_1 = 0$

$$\boxed{\pi_1 = \theta}$$

> $\theta$ 本身就是无量纲量，自然成为一个 $\pi$ 项。

**$\pi_2$（含 $T$）**：

$$\pi_2 = T \cdot l^{a_2} g^{b_2} m^{c_2}$$

$$[\pi_2] = T^1 \cdot L^{a_2} \cdot L^{b_2} T^{-2b_2} \cdot M^{c_2} = M^{c_2} L^{a_2+b_2} T^{1-2b_2}$$

- $c_2 = 0$
- $1 - 2b_2 = 0 \Rightarrow b_2 = 1/2$
- $a_2 + b_2 = 0 \Rightarrow a_2 = -1/2$

$$\boxed{\pi_2 = T \cdot l^{-1/2} \cdot g^{1/2} = T\sqrt{\frac{g}{l}}}$$

**Step 5**：$n - k = 5 - 3 = 2$ 个 $\pi$ 项

$$f\!\left(\theta,\; T\sqrt{\frac{g}{l}}\right) = 0$$

$$\boxed{T = \sqrt{\frac{l}{g}} \cdot \phi(\theta)}$$

**结论**：
- 周期**与质量 $m$ 无关**（$m$ 不出现在任何 $\pi$ 项中！）
- 对小角度 $\theta \ll 1$，$\phi(\theta) \approx 2\pi$，得到 $T = 2\pi\sqrt{l/g}$

### 例 2：流体中运动物体的阻力

**问题**：一个球在粘性流体中运动，受到的阻力 $F_D$ 与哪些量有关？

**Step 1**：相关物理量

| 量 | 符号 | 量纲 |
|:---:|:---:|:---:|
| 阻力 | $F_D$ | $M L T^{-2}$ |
| 球直径 | $d$ | $L$ |
| 速度 | $v$ | $L T^{-1}$ |
| 流体密度 | $\rho$ | $M L^{-3}$ |
| 流体粘度 | $\mu$ | $M L^{-1} T^{-1}$ |

$n = 5$，$k = 3$（$M, L, T$），应有 $5 - 3 = 2$ 个 $\pi$ 项。

**Step 2**：选取重复量 $d, v, \rho$

**Step 3**：构造 $\pi$ 项

**$\pi_1$（含 $\mu$）**：

$$\pi_1 = \mu \cdot d^{a_1} v^{b_1} \rho^{c_1}$$

$$[\pi_1] = M L^{-1} T^{-1} \cdot L^{a_1} \cdot (L T^{-1})^{b_1} \cdot (M L^{-3})^{c_1}$$

$$= M^{1+c_1} \cdot L^{-1+a_1+b_1-3c_1} \cdot T^{-1-b_1}$$

- $1 + c_1 = 0 \Rightarrow c_1 = -1$
- $-1 - b_1 = 0 \Rightarrow b_1 = -1$
- $-1 + a_1 + b_1 - 3c_1 = 0 \Rightarrow -1 + a_1 - 1 + 3 = 0 \Rightarrow a_1 = -1$

$$\boxed{\pi_1 = \frac{\mu}{\rho v d} = \text{Re}^{-1}}$$

其中 $\text{Re} = \dfrac{\rho v d}{\mu}$ 是 **Reynolds 数**。

**$\pi_2$（含 $F_D$）**：

$$\pi_2 = F_D \cdot d^{a_2} v^{b_2} \rho^{c_2}$$

$$= M L T^{-2} \cdot L^{a_2} \cdot (L T^{-1})^{b_2} \cdot (M L^{-3})^{c_2}$$

$$= M^{1+c_2} \cdot L^{1+a_2+b_2-3c_2} \cdot T^{-2-b_2}$$

- $1 + c_2 = 0 \Rightarrow c_2 = -1$
- $-2 - b_2 = 0 \Rightarrow b_2 = -2$
- $1 + a_2 + b_2 - 3c_2 = 0 \Rightarrow 1 + a_2 - 2 + 3 = 0 \Rightarrow a_2 = -2$

$$\boxed{\pi_2 = \frac{F_D}{\rho v^2 d^2} = C_D}$$

其中 $C_D$ 是 **阻力系数**。

**Step 4**：最终关系

$$\boxed{C_D = f(\text{Re})}$$

即：

$$F_D = \frac{1}{2} C_D(\text{Re}) \cdot \rho v^2 \cdot \frac{\pi d^2}{4}$$

> 这就是流体力学中著名的**阻力公式**！函数 $C_D(\text{Re})$ 需要通过实验测定：
> - 低 Re（层流）：$C_D = 24/\text{Re}$（Stokes 定律）
> - 高 Re（湍流）：$C_D \approx 0.44$（Newton 阻力区）

### 例 3：原子弹爆炸的火球半径（Taylor 问题）

**问题**：原子弹爆炸后，火球半径 $R$ 与哪些量有关？

**物理学家 G.I. Taylor（1950）** 用此方法从公开照片推算出了核弹当量（当时是机密）。

**Step 1**：相关物理量

|  量   |    符号    |       量纲       |
| :--: | :------: | :------------: |
| 火球半径 |   $R$    |      $L$       |
| 爆炸能量 |   $E$    | $M L^2 T^{-2}$ |
|  时间  |   $t$    |      $T$       |
| 空气密度 | $\rho_0$ |   $M L^{-3}$   |

$n = 4$，$k = 3$，应有 $4 - 3 = 1$ 个 $\pi$ 项。

**Step 2**：选取重复量 $E, t, \rho_0$

**Step 3**：

$$\pi = R \cdot E^{a} t^{b} \rho_0^{c}$$

$$= L \cdot (M L^2 T^{-2})^{a} \cdot T^{b} \cdot (M L^{-3})^{c}$$

$$= M^{a+c} \cdot L^{1+2a-3c} \cdot T^{-2a+b}$$

- $a + c = 0 \Rightarrow c = -a$
- $-2a + b = 0 \Rightarrow b = 2a$
- $1 + 2a - 3c = 0 \Rightarrow 1 + 2a + 3a = 0 \Rightarrow a = -1/5$

$$a = -\frac{1}{5}, \quad b = -\frac{2}{5}, \quad c = \frac{1}{5}$$

$$\boxed{\pi = R \left(\frac{\rho_0}{E t^2}\right)^{1/5} = \text{const}}$$

即：

$$\boxed{R = \xi \left(\frac{E}{\rho_0}\right)^{1/5} t^{2/5}}$$

> **Taylor 的天才操作**：他对核爆高速摄影照片测量 $R(t)$，发现 $R \propto t^{2/5}$ 的标度律完美成立！从斜率和空气密度 $\rho_0$ 反推出能量 $E$，从而推算出了核弹当量——这在当时是绝密信息。

### 例 4：弦的振动频率

**问题**：一根张紧的弦，其振动基频 $f$ 与哪些量有关？

**Step 1**：

| 量 | 符号 | 量纲 |
|:---:|:---:|:---:|
| 频率 | $f$ | $T^{-1}$ |
| 弦长 | $l$ | $L$ |
| 张力 | $F$ | $M L T^{-2}$ |
| 线密度 | $\mu$ | $M L^{-1}$ |

$n = 4$，$k = 3$，应有 $1$ 个 $\pi$ 项。

**Step 2**：重复量 $l, F, \mu$

$$\pi = f \cdot l^{a} F^{b} \mu^{c}$$

$$= T^{-1} \cdot L^{a} \cdot (M L T^{-2})^{b} \cdot (M L^{-1})^{c}$$

$$= M^{b+c} \cdot L^{a+b-c} \cdot T^{-1-2b}$$

- $b + c = 0 \Rightarrow c = -b$
- $-1 - 2b = 0 \Rightarrow b = -1/2$
- $a + b - c = 0 \Rightarrow a - 1/2 - (-1/2) = 0 \Rightarrow a = 0$... 

等等，$a + b - c = a + (-1/2) - (1/2) = a - 1 = 0 \Rightarrow a = 1$

$$\pi = f \cdot l^1 \cdot F^{-1/2} \cdot \mu^{1/2} = f l \sqrt{\frac{\mu}{F}}$$

$$\boxed{f = \frac{\xi}{l}\sqrt{\frac{F}{\mu}}}$$

> 与精确解 $f = \dfrac{1}{2l}\sqrt{F/\mu}$ 完全一致（$\xi = 1/2$ 需要理论或实验确定）。

## 五、量纲矩阵方法（系统化方法）

对于复杂问题，可以用**线性代数**系统化地构造 $\pi$ 项。

### 方法

设有 $n$ 个物理量，$k$ 个基本量纲。构造 $k \times n$ 的**量纲矩阵** $\mathbf{A}$：

$$\mathbf{A} = \begin{pmatrix} [q_1]_M & [q_2]_M & \cdots & [q_n]_M \\ [q_1]_L & [q_2]_L & \cdots & [q_n]_L \\ [q_1]_T & [q_2]_T & \cdots & [q_n]_T \end{pmatrix}$$

每个 $\pi$ 项对应量纲矩阵的一个**零空间向量** $\mathbf{x}$：

$$\mathbf{A} \mathbf{x} = \mathbf{0}$$

$\pi$ 项的个数 = $n - \text{rank}(\mathbf{A})$。

### 示例：流体阻力问题

$$\mathbf{A} = \begin{pmatrix} F_D & d & v & \rho & \mu \\ \hline 1 & 0 & 0 & 1 & 1 \\ 1 & 1 & 1 & -3 & -1 \\ -2 & 0 & -1 & 0 & -1 \end{pmatrix}$$

$\text{rank}(\mathbf{A}) = 3$，零空间维数 $= 5 - 3 = 2$，与前面的结论一致。

## 六、$\pi$ 定理的深层意义与局限

### 意义

| 方面         | 说明                              |
| :--------- | :------------------------------ |
| **减少实验量**  | 5 个变量的函数 → 2 个无量纲量的函数，实验工作量大幅减少 |
| **指导模型实验** | 只要保证 $\pi$ 项相同（相似准则），小模型可以模拟大系统 |
| **无需求解方程** | 不需要知道控制方程的具体形式，只需知道参与的物理量       |
| **检验公式**   | 任何正确公式的每一项必须量纲一致                |

### 局限

| 局限 | 说明 |
|:---|:---|
| **不能确定函数形式** | 只能给出 $f(\pi_1, \pi_2) = 0$，无法知道 $f$ 是线性、指数还是其他形式 |
| **依赖物理直觉** | 漏选或多选物理量会导致错误结论 |
| **不能确定无量纲常数** | 如单摆中的 $2\pi$，需要理论或实验补充 |
| **不能处理有量纲常数** | 如果问题涉及一个有量纲的普适常数（如 $\hbar, G$），必须将其纳入分析 |

## 七、相似准则与模型实验

$\pi$ 定理在工程中最强大的应用是**模型实验**：

> **如果原型和模型的所有 $\pi$ 项相等，则两者物理行为相似。**

### 经典矛盾：Froude 相似 vs Reynolds 相似

船舶阻力问题同时涉及：
- **Froude 数** $\text{Fr} = v/\sqrt{gl}$（重力效应）
- **Reynolds 数** $\text{Re} = \rho v l / \mu$（粘性效应）

若要同时满足 $\text{Fr}_{\text{原型}} = \text{Fr}_{\text{模型}}$ 和 $\text{Re}_{\text{原型}} = \text{Re}_{\text{模型}}$，则要求：

$$\frac{v_m}{v_p} = \sqrt{\frac{l_m}{l_p}} \quad (\text{Fr 相似})$$

$$\frac{v_m}{v_p} = \frac{l_p}{l_m} \quad (\text{Re 相似})$$

除非 $l_m = l_p$（不是模型），否则**不可能同时满足**。这就是著名的**相似律矛盾**，工程上需要根据主导效应做取舍。

## 八、总结

$$\boxed{\text{Buckingham } \pi \text{ 定理：} \quad n \text{ 个量，} k \text{ 个基本量纲} \;\Longrightarrow\; (n-k) \text{ 个独立无量纲量}}$$

| 步骤  | 关键操作                                                                   |
| :-: | :--------------------------------------------------------------------- |
|  ①  | 列出所有相关物理量（物理直觉！）                                                       |
|  ②  | 写出量纲矩阵，确定 $k = \text{rank}(\mathbf{A})$                                |
|  ③  | 选取 $k$ 个线性无关的重复量                                                       |
|  ④  | 对每个非重复量，令 $\pi = q_i \cdot q_{r_1}^{a_1}\cdots q_{r_k}^{a_k}$ 无量纲，解出指数 |
|  ⑤  | 写出 $f(\pi_1, \ldots, \pi_{n-k}) = 0$                                   |

> **量纲分析是物理学家的"第六感"**——即使不知道方程的细节，也能从中提取出深刻的标度律和相似关系。Taylor 用它推算核弹当量的故事，就是这种力量最精彩的证明。
