## 一、引入动机

在非相对论量子力学中，角动量 $\vec{J}$ 是一个良定义的守恒量，其平方 $\hat{J}^2$ 的本征值 $j(j+1)$ 直接标记了粒子的角动量量子数。

但当我们进入**相对论性**理论时，情况变得复杂了：

- 时空对称群从伽利略群升级为**庞加莱群**，其生成元包括四动量 $P^\mu$ 和洛伦兹生成元 $J^{\mu\nu}$（共 10 个）。
- 空间旋转只是洛伦兹变换的一个子集，**轨道角动量与自旋角动量的分离不再洛伦兹协变**。
- 我们需要一个**洛伦兹协变**的对象来刻画粒子的自旋。

这正是 Pauli-Lubanski 赝矢量所扮演的角色——它是**自旋在相对论框架下的自然推广**。

---

## 二、数学定义

### 1. 洛伦兹生成元

庞加莱群的洛伦兹部分有 6 个生成元 $J^{\mu\nu}$（反对称：$J^{\mu\nu} = -J^{\nu\mu}$），其中：
- $J^{ij} = \epsilon^{ijk}J^k$：3 个空间旋转生成元（$i,j,k = 1,2,3$）
- $K^i = J^{0i}$：3 个推进（boost）生成元

满足洛伦兹代数：

$$[J^{\mu\nu}, J^{\rho\sigma}] = i(\eta^{\mu\rho}J^{\nu\sigma} - \eta^{\mu\sigma}J^{\nu\rho} - \eta^{\nu\rho}J^{\mu\sigma} + \eta^{\nu\sigma}J^{\mu\rho})$$

### 2. Pauli-Lubanski 赝矢量的定义

$$\boxed{W^\mu = \frac{1}{2}\,\epsilon^{\mu\nu\rho\sigma}\,P_\nu\, J_{\rho\sigma}}$$

其中：
- $\epsilon^{\mu\nu\rho\sigma}$ 是四维 Levi-Civita 全反对称张量（$\epsilon^{0123} = +1$）
- $P_\nu$ 是四动量算符
- $J_{\rho\sigma}$ 是洛伦兹生成元

展开写成分量形式（采用度规 $\eta = \mathrm{diag}(+,-,-,-)$）：

$$W^0 = \frac{1}{2}\epsilon^{0\nu\rho\sigma}P_\nu J_{\rho\sigma} = P_i J^{0i} \cdot (\text{组合})$$

更明确地，利用 $\epsilon^{0ijk} = \epsilon^{ijk}$：

$$W^0 = \vec{P} \cdot \vec{J}$$

$$W^i = P^0 J^i - \epsilon^{ijk}P^j K^k$$

其中 $J^i = \frac{1}{2}\epsilon^{ijk}J^{jk}$ 是空间旋转生成元，$K^k = J^{0k}$ 是推进生成元。

用三维矢量紧凑地写：

$$\boxed{W^0 = \vec{P}\cdot\vec{J}\,, \qquad \vec{W} = E\,\vec{J} - \vec{P}\times\vec{K}}$$

> $W^\mu$ 本质上是把三维角动量 $\vec{J}$ "嵌入"到了四维时空中，同时用四动量 $P^\mu$ 和推进生成元 $\vec{K}$ 来保证整个对象在洛伦兹变换下有良好的协变性质。

## 三、基本性质与代数结构

### 性质 1：与四动量正交

$$\boxed{W^\mu P_\mu = 0}$$

**证明**：

$$W^\mu P_\mu = \frac{1}{2}\epsilon^{\mu\nu\rho\sigma}P_\nu J_{\rho\sigma} P_\mu = \frac{1}{2}\epsilon^{\mu\nu\rho\sigma}P_\mu P_\nu J_{\rho\sigma}$$

由于 $\epsilon^{\mu\nu\rho\sigma}$ 对 $\mu,\nu$ 反对称，而 $P_\mu P_\nu$ 对 $\mu,\nu$ 对称，因此缩并为零。

> 这意味着 $W^\mu$ 在粒子的静止系中只有三个独立分量（$W^0 = 0$），正如我们预期的"三维角动量"。

### 性质 2：与四动量对易

$$\boxed{[W^\mu, P^\nu] = 0}$$

这表明 $W^\mu$ 是庞加莱群的**卡西米尔型**算符的一部分——它与所有平移对易，因此在粒子的任意态上都有确定值。

### 性质 3：$W^\mu$ 之间的对易关系

$$\boxed{[W^\mu, W^\nu] = -i\,\epsilon^{\mu\nu}{}_{\rho\sigma}\,W^\rho\, P^\sigma}$$

这个代数结构非常特殊——它**不是**封闭的李代数，因为右边出现了 $P^\sigma$。但在给定 $P^\mu$ 的本征值（即固定了粒子的四动量）之后，$W^\mu$ 之间的对易关系就封闭了，构成一个子代数。

### 性质 4：$W^\mu$ 是赝矢量

从定义中 Levi-Civita 张量的存在可以看出，$W^\mu$ 在空间反演（宇称 $P$）下的行为是：

$$P:\quad W^0 \to W^0,\quad \vec{W} \to -\vec{W}$$

即 $W^\mu$ 是一个**赝矢量**（pseudovector），与普通角动量的宇称性质一致。

## 四、第二个卡西米尔算符与自旋

庞加莱群有两个独立的卡西米尔不变量：

| 卡西米尔算符 | 定义 | 物理意义 |
|---|---|---|
| $\mathcal{C}_1$ | $P^\mu P_\mu$ | 质量平方：$m^2$ |
| $\mathcal{C}_2$ | $W^\mu W_\mu$ | 自旋：$-m^2 s(s+1)$ |

### 在静止系中计算 $W^\mu W_\mu$

对于**有质量粒子**（$m > 0$），可以选择**静止系**（rest frame），此时：

$$P^\mu = (m, \vec{0})$$

代入 $W^\mu$ 的分量表达式：

$$W^0 = \vec{P}\cdot\vec{J} = 0$$
$$W^i = E\,J^i - (\vec{P}\times\vec{K})^i = m\,J^i$$

因此：

$$W^\mu W_\mu = -(W^i)^2 = -m^2 \vec{J}^2$$

由于 $\vec{J}^2$ 的本征值为 $j(j+1)$（这里 $j = s$，即自旋量子数）：

$$\boxed{W^\mu W_\mu = -m^2\, s(s+1)}$$

> 在粒子"静止不动"的参考系中，$W^\mu$ 的空间分量恰好就是 $m$ 乘以通常的角动量算符。$W^\mu W_\mu$ 是洛伦兹不变量，所以它在**任何**参考系中都等于 $-m^2 s(s+1)$——这就是自旋作为粒子内禀属性的洛伦兹不变刻画。

## 五、Wigner 分类：$W^\mu$ 如何标记粒子

Eugene Wigner 在 1939 年利用 $W^\mu$ 对庞加莱群的不可约酉表示进行了完整分类，这是**粒子物理中"粒子"概念的数学基础**。

### 情形 A：有质量粒子（$m > 0$）

在静止系中，$P^\mu = (m, \vec{0})$，稳定子群（little group）是 $\mathrm{SO}(3)$。

$W^\mu$ 约化为：

$$\frac{W^i}{m} = J^i$$

这正是标准的 $\mathrm{SO}(3)$ 角动量代数。不可约表示由 $s = 0, \frac{1}{2}, 1, \ldots$ 标记，每个表示有 $2s+1$ 个态 $|s, m_s\rangle$（$m_s = -s, \ldots, +s$）。

> 粒子的种类由 $(m, s)$ 完全标记（忽略内部量子数时）。

### 情形 B：无质量粒子（$m = 0$）

此时**不存在静止系**，不能直接约化到 $\mathrm{SO}(3)$。取标准动量 $P^\mu = (E, 0, 0, E)$（沿 $z$ 方向），稳定子群是 $\mathrm{ISO}(2)$（二维欧几里得群）。

$W^\mu$ 在此情形下满足：

$$W^\mu = h\, P^\mu$$

其中 $h$ 是**螺旋度**（helicity）：

$$h = \frac{\vec{J}\cdot\hat{p}}{|\vec{p}|}$$

$h$ 是洛伦兹不变量（对于无质量粒子），且取值 $h = 0, \pm\frac{1}{2}, \pm 1, \ldots$

关键区别：无质量粒子只有**两个**螺旋度态 $h = \pm s$（而非 $2s+1$ 个）。例如：
- 光子（$s=1$）：$h = \pm 1$，无 $h = 0$ 的纵向极化态
- 引力子（假设 $s=2$）：$h = \pm 2$

> 对于无质量粒子，你永远追不上它（没有静止系），所以你无法看到它的所有自旋投影——只能观测到沿运动方向的投影（螺旋度）。这就是为什么光子只有两种偏振态，而不是三种。

## 六、各分量的物理意义总结

| 分量 | 表达式 | 物理意义 |
|------|--------|----------|
| $W^0$ | $\vec{P}\cdot\vec{J}$ | 角动量在动量方向的投影（与螺旋度相关） |
| $\vec{W}$ | $E\vec{J} - \vec{P}\times\vec{K}$ | 角动量的"洛伦兹协变推广"，混合了旋转和推进 |
| $W^\mu W_\mu$ | $-m^2 s(s+1)$ | 洛伦兹不变量，直接给出自旋 |
| $W^\mu / m$（静止系） | $(0, \vec{J})$ | 退化为通常的三维角动量 |

---

## 七、一个具体的例子：电子

电子是自旋 $s = \frac{1}{2}$、质量 $m_e$ 的粒子。

$$W^\mu W_\mu = -m_e^2 \cdot \frac{1}{2}\cdot\frac{3}{2} = -\frac{3}{4}m_e^2$$

在静止系中：
- $W^0 = 0$
- $\vec{W} = m_e \vec{S} = m_e \cdot \frac{\hbar}{2}\vec{\sigma}$

自旋在任意方向的投影只能取 $\pm\frac{\hbar}{2}$，对应 $2s+1 = 2$ 个态。

当电子运动时，$W^\mu$ 的分量会发生"混合"——空间旋转和推进耦合在一起——但 $W^\mu W_\mu$ 始终不变，保证了自旋的洛伦兹不变性。

## 八、总结

| 要点 | 内容 |
|------|------|
| **定义** | $W^\mu = \frac{1}{2}\epsilon^{\mu\nu\rho\sigma}P_\nu J_{\rho\sigma}$ |
| **本质** | 角动量的洛伦兹协变推广 |
| **核心性质** | $W^\mu P_\mu = 0$，$[W^\mu, P^\nu] = 0$ |
| **物理意义** | $W^\mu W_\mu = -m^2 s(s+1)$ 是自旋的洛伦兹不变刻画 |
| **有质量粒子** | 静止系中 $\vec{W}/m = \vec{J}$，稳定子群 $\mathrm{SO}(3)$ |
| **无质量粒子** | $W^\mu = hP^\mu$，螺旋度 $h$ 标记表示，稳定子群 $\mathrm{ISO}(2)$ |
| **历史意义** | Wigner (1939) 用此完成了庞加莱群表示的分类——奠定了"粒子"概念的数学基础 |
