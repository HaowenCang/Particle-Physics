## 一、核心思想：局域对称性决定相互作用

规范耦合的本质源于一个深刻的原则：

> **要求物理系统的拉格朗日量在局域规范变换下保持不变，就必须引入规范场，而规范场与物质场之间的相互作用强度由规范耦合常数 $g$ 来刻画。**

这是 Yang-Mills 理论（1954）的核心精神，也是整个标准模型的构造基础。

---

## 二、从全局对称性到局域对称性

### 2.1 全局对称性

考虑一个自由狄拉克费米子场 $\psi(x)$，其拉格朗日量为：

$$\mathcal{L}_0 = \bar{\psi}(i\gamma^\mu\partial_\mu - m)\psi$$

此拉格朗日量在**全局** $U(1)$ 变换下不变：

$$\psi(x) \;\to\; e^{i\alpha}\,\psi(x), \quad \alpha = \text{常数}$$

因为 $\partial_\mu\psi$ 同样乘以 $e^{i\alpha}$，整体相位在 $\bar{\psi}(\cdots)\psi$ 中抵消。

### 2.2 局域化——问题的产生

如果要求对称性是**局域**的，即：

$$\psi(x) \;\to\; e^{i\alpha(x)}\,\psi(x)$$

则导数项不再协变：

$$\partial_\mu\psi \;\to\; \partial_\mu\!\left(e^{i\alpha(x)}\psi\right) = e^{i\alpha(x)}\left(\partial_\mu\psi + i(\partial_\mu\alpha)\psi\right)$$

多出的 $i(\partial_\mu\alpha)\psi$ 项破坏了规范不变性。

> **通俗理解**：全局相位变换像是把所有时钟同时拨快一个固定值，没有可观测效应；局域变换则像是每个时空点的时钟拨快不同值，导数项（衡量相邻点的差异）就会"感知"到这种不协调。

### 2.3 协变导数——引入规范场

为了恢复不变性，将普通导数 $\partial_\mu$ 替换为**协变导数**：

$$D_\mu = \partial_\mu + igA_\mu$$

其中 $A_\mu$ 是新引入的**规范场**（电磁场的推广），$g$ 就是**规范耦合常数**。

要求 $D_\mu\psi$ 与 $\psi$ 以相同方式变换：

$$D_\mu\psi \;\to\; e^{i\alpha(x)}\,D_\mu\psi$$

这迫使规范场做如下变换：

$$A_\mu \;\to\; A_\mu - \frac{1}{g}\,\partial_\mu\alpha$$

这正是电磁势的规范变换 $A_\mu \to A_\mu - \partial_\mu\Lambda$ 的推广（取 $\alpha = g\Lambda$）。

### 2.4 相互作用的涌现

将 $\partial_\mu \to D_\mu$ 代入拉格朗日量：

$$\mathcal{L} = \bar{\psi}(i\gamma^\mu D_\mu - m)\psi = \bar{\psi}(i\gamma^\mu\partial_\mu - m)\psi \;-\; g\,\bar{\psi}\gamma^\mu\psi\,A_\mu$$

最后一项：

$$\boxed{\mathcal{L}_{\text{int}} = -g\,\bar{\psi}\gamma^\mu\psi\,A_\mu}$$

这就是**规范相互作用项**。它不是人为添加的，而是**局域对称性的必然后果**。

> **通俗理解**：要求物理定律在每个时空点独立地做相位旋转时保持不变，就"逼迫"宇宙必须存在一个传递相互作用的场（规范场），而 $g$ 决定了物质粒子与这个场"握手"的力度。

## 三、非阿贝尔规范理论（Yang-Mills 理论）

### 3.1 从 $U(1)$ 到非阿贝尔群

当规范群是**非阿贝尔**（不可交换）的，例如 $\mathrm{SU(N)}$，情况变得更加丰富。

设规范群的生成元为 $T^a$（$a = 1, \ldots, N^2-1$），满足李代数：

$$[T^a, T^b] = if^{abc}\,T^c$$

其中 $f^{abc}$ 是**结构常数**。

物质场（如夸克）在规范群的表示下变换：

$$\psi(x) \;\to\; U(x)\,\psi(x) = e^{i\alpha^a(x)T^a}\,\psi(x)$$

协变导数推广为：

$$D_\mu = \partial_\mu + ig\,A_\mu^a\,T^a$$

规范场的变换规则为：

$$A_\mu^a\,T^a \;\to\; U\!\left(A_\mu^a\,T^a\right)\!U^{-1} + \frac{i}{g}\,U\,\partial_\mu U^{-1}$$

### 3.2 规范场的自相互作用

与 $U(1)$ 情形的关键区别：**非阿贝尔规范场自身携带"荷"，因此与自身发生相互作用。**

规范场强张量为：

$$F_{\mu\nu}^a = \partial_\mu A_\nu^a - \partial_\nu A_\mu^a - g\,f^{abc}\,A_\mu^b\,A_\nu^c$$

Yang-Mills 拉格朗日量：

$$\mathcal{L}_{\text{YM}} = -\frac{1}{4}\,F_{\mu\nu}^a\,F^{a\,\mu\nu}$$

展开后包含：

| 项 | 物理含义 |
|----|----------|
| $(\partial A)^2$ | 规范场的动能项（传播子） |
| $g\,(\partial A)\,A\,A$ | **三胶子顶点**（三线耦合） |
| $g^2\,A\,A\,A\,A$ | **四胶子顶点**（四线耦合） |

> **通俗理解**：电磁场（光子）不带电荷，所以光子之间不直接相互作用。但胶子携带色荷，所以胶子之间可以"粘在一起"——这是 QCD 独特性质（渐近自由、色禁闭）的根源。

## 四、标准模型中的三种规范耦合

标准模型的规范群为：

$$G_{\text{SM}} = \mathrm{SU(3)_C \times SU(2)_L \times U(1)_Y}$$

对应三种规范耦合：

### 4.1 强相互作用：$\mathrm{SU(3)_C}$

- **规范场**：8 种胶子 $G_\mu^a$（$a=1,\ldots,8$）
- **耦合常数**：$g_s$
- **耦合对象**：所有带色荷的粒子（夸克、胶子本身）
- **拉格朗日量**：

$$\mathcal{L}_{\text{QCD}} = \bar{q}\!\left(i\gamma^\mu D_\mu - m_q\right)\!q - \frac{1}{4}\,G_{\mu\nu}^a\,G^{a\,\mu\nu}$$

$$D_\mu = \partial_\mu + ig_s\,G_\mu^a\,T^a$$

- **费曼图顶点**：夸克-夸克-胶子（$g_s$），三胶子（$g_s$），四胶子（$g_s^2$）

### 4.2 弱相互作用：$\mathrm{SU(2)_L}$

- **规范场**：$W_\mu^1, W_\mu^2, W_\mu^3$（3 种）
- **耦合常数**：$g$（或 $g_2$）
- **耦合对象**：所有左手费米子二重态
- **关键特征**：只与**左手**粒子耦合——这是弱相互作用**宇称不守恒**的根源

### 4.3 超弱相互作用：$\mathrm{U(1)_Y}$

- **规范场**：$B_\mu$（1 种）
- **耦合常数**：$g'$（或 $g_1$）
- **耦合对象**：所有具有超荷 $Y$ 的粒子
- **关键特征**：$B_\mu$ 与右手和左手费米子均耦合（但耦合强度不同）

### 4.4 电弱混合与 Weinberg 角

Higgs 机制破缺 $\mathrm{SU(2)_L \times U(1)_Y} \to \mathrm{U(1)_{EM}}$ 后，规范场发生混合：

$$\begin{pmatrix} A_\mu \\ Z_\mu \end{pmatrix} = \begin{pmatrix} \cos\theta_W & \sin\theta_W \\ -\sin\theta_W & \cos\theta_W \end{pmatrix} \begin{pmatrix} B_\mu \\ W_\mu^3 \end{pmatrix}$$

其中 **Weinberg 角** $\theta_W$ 由两个耦合常数的比值决定：

$$\tan\theta_W = \frac{g'}{g}$$

物理的规范玻色子为：
- **光子** $A_\mu$：无质量，耦合强度 $e = g\sin\theta_W = g'\cos\theta_W$
- $W^\pm_\mu = \frac{1}{\sqrt{2}}(W_\mu^1 \mp iW_\mu^2)$：有质量
- $Z_\mu$：有质量

## 五、规范耦合常数的跑动（Running）

这是规范耦合最深刻的特征之一。

### 5.1 重整化群方程

在量子水平上，规范耦合常数不是真正的"常数"，而是**随能量标度 $\mu$ 变化**的函数，满足 $\beta$ 函数方程：

$$\mu\frac{dg}{d\mu} = \beta(g)$$

对于 $\mathrm{SU(N)}$ 规范理论中 $N_f$ 种费米子，单圈 $\beta$ 函数为：

$$\beta(g) = -\frac{g^3}{16\pi^2}\left(\frac{11}{3}N - \frac{2}{3}N_f\right)$$

括号中两项的竞争：
- $\frac{11}{3}N$：来自规范场自相互作用（胶子圈），使耦合**减弱**（屏蔽效应的反面——**反屏蔽**）
- $-\frac{2}{3}N_f$：来自费米子圈（真空极化），使耦合**增强**（正常屏蔽）

### 5.2 渐近自由与色禁闭

对于 QCD（$N=3$），只要 $N_f < 33/2 = 16.5$（标准模型中 $N_f = 6$），就有：

$$\beta(g_s) < 0$$

这意味着：

$$\boxed{\alpha_s(\mu) = \frac{g_s^2(\mu)}{4\pi} \;\xrightarrow{\mu\to\infty}\; 0}$$

- **高能（短距离）**：耦合变弱，夸克表现得像自由粒子——**渐近自由**（Gross, Wilczek, Politzer，2004年诺贝尔奖）
- **低能（长距离）**：耦合变强，夸克被束缚在强子内部——**色禁闭**

> **通俗理解**：想象一根橡皮筋，拉得越长（距离越大）张力越强，你永远无法把夸克拉出来单独观察。这与电磁力的行为恰好相反。

### 5.3 三种耦合常数的跑动行为

用 $\alpha_i = g_i^2/(4\pi)$ 表示：

| 规范群 | $\beta$ 函数符号 | 行为 |
|--------|-----------------|------|
| $\mathrm{SU(3)_C}$ | $\beta < 0$ | 随能量升高而**减小**（渐近自由） |
| $\mathrm{SU(2)_L}$ | $\beta > 0$（标准模型） | 随能量升高而**增大** |
| $\mathrm{U(1)_Y}$ | $\beta > 0$ | 随能量升高而**增大** |

三条曲线在 $M_Z \approx 91\,\text{GeV}$ 附近的实验值为：

$$\alpha_s(M_Z) \approx 0.118, \quad \alpha_2(M_Z) \approx 0.034, \quad \alpha_1(M_Z) \approx 0.017$$

> **大统一理论（GUT）的线索**：在标准模型中，三条跑动曲线在高能处**近似交汇但不精确交于一点**。引入**超对称**（SUSY）后，它们在 $\sim 10^{16}\,\text{GeV}$ 处精确交汇，这被视为大统一理论的重要间接证据。

## 六、规范耦合与费曼规则

在微扰论计算中，规范耦合常数直接出现在**费曼图的顶点因子**中：

| 顶点 | 耦合因子 | 对应物理过程 |
|------|---------|-------------|
| 夸克-夸克-胶子 | $ig_s\,\gamma^\mu\,T^a$ | 夸克散射 |
| 三胶子 | $g_s\,f^{abc}$ | 胶子自相互作用 |
| 费米子-$W$-$\nu$ | $\frac{ig}{2\sqrt{2}}\,\gamma^\mu(1-\gamma^5)$ | $\beta$ 衰变 |
| 费米子-光子 | $ieQ_f\,\gamma^\mu$ | 康普顿散射 |

散射截面的领头阶通常正比于耦合常数的平方：

$$\sigma \propto \alpha = \frac{g^2}{4\pi}$$

高阶修正按 $\alpha/\pi$ 的幂次展开，这就是**微扰展开的小参数**。

## 七、概念总结

将规范耦合的核心逻辑用一句话概括：

$$\boxed{\text{局域规范不变性} \;\Longrightarrow\; \text{规范场的必要性} \;\Longrightarrow\; \text{规范耦合}g\,\text{决定相互作用强度}}$$

| 概念 | 含义 |
|------|------|
| **规范对称性** | 物理定律在局域变换下的不变性 |
| **协变导数** | $D_\mu = \partial_\mu + igA_\mu^a T^a$，引入规范场 |
| **规范耦合常数 $g$** | 物质场与规范场相互作用的强度 |
| **规范场自耦合** | 非阿贝尔理论中规范场携带"荷"，与自身相互作用 |
| **跑动** | $g(\mu)$ 随能量标度变化，由 $\beta$ 函数控制 |
| **渐近自由 / 色禁闭** | QCD 耦合常数跑动的直接物理后果 |
