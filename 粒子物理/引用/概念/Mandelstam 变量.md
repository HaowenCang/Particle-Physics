## 一、历史背景

### 1.1 问题的起源

在 20 世纪 50 年代，粒子物理散射理论面临一个核心问题：

> **如何用最少的、洛伦兹不变的变量来完整描述一个散射过程的运动学？**

对于 $2 \to 2$ 散射过程，初态有两个粒子、末态有两个粒子，共涉及四个四维动量 $p_1, p_2, p_3, p_4$。这些动量的分量共有 $4 \times 4 = 16$ 个，但受到以下约束：

- 四维动量守恒：$p_1 + p_2 = p_3 + p_4$（4 个条件）
- 质壳条件：$p_i^2 = -m_i^2$（4 个条件）
- 洛伦兹不变性：消除参考系选择的自由度（6 个条件）

独立运动学变量的个数：

$$16 - 4 - 4 - 6 = 2$$

即**只需两个独立的洛伦兹不变量**即可完全确定 $2 \to 2$ 散射的运动学。

### 1.2 Mandelstam 的贡献

1958 年，南非裔美国物理学家 **Stanley Mandelstam** 在研究散射振幅的解析性质时，引入了一组对称的洛伦兹不变量——后来被称为 **Mandelstam 变量**。这一工作不仅简化了散射运动学的描述，更为散射振幅的解析结构、交叉对称性和色散关系奠定了数学基础。

---

## 二、定义

### 2.1 四维动量的约定

考虑 $2 \to 2$ 散射过程：

$$1 + 2 \;\to\; 3 + 4$$

四维动量记为 $p_1, p_2$（入射）和 $p_3, p_4$（出射），采用度规约定 $\eta_{\mu\nu} = \mathrm{diag}(-1,+1,+1,+1)$。

四维动量守恒：

$$p_1 + p_2 = p_3 + p_4$$

### 2.2 Mandelstam 变量的定义

$$\boxed{
\begin{aligned}
s &= -(p_1 + p_2)^2 = -(p_3 + p_4)^2 \\[6pt]
t &= -(p_1 - p_3)^2 = -(p_2 - p_4)^2 \\[6pt]
u &= -(p_1 - p_4)^2 = -(p_2 - p_3)^2
\end{aligned}
}$$

> **物理含义**：
> - $s$：初态（或末态）总四维动量的模方——**质心系总能量的平方**
> - $t$：初态粒子 1 与末态粒子 3 之间的四维动量转移的模方——**动量转移平方**
> - $u$：初态粒子 1 与末态粒子 4 之间的四维动量转移的模方——**交叉动量转移平方**

---

## 三、基本性质

### 3.1 质量关系（Mandelstam 恒等式）

三个 Mandelstam 变量不是完全独立的。利用质壳条件 $p_i^2 = -m_i^2$ 和动量守恒，可以证明：

$$\boxed{s + t + u = m_1^2 + m_2^2 + m_3^2 + m_4^2}$$

**证明**：

$$s + t + u = -(p_1+p_2)^2 - (p_1-p_3)^2 - (p_1-p_4)^2$$

展开每一项：

$$-(p_1+p_2)^2 = -p_1^2 - p_2^2 - 2p_1\cdot p_2 = m_1^2 + m_2^2 + 2p_1\cdot p_2$$

$$-(p_1-p_3)^2 = -p_1^2 - p_3^2 + 2p_1\cdot p_3 = m_1^2 + m_3^2 - 2p_1\cdot p_3$$

$$-(p_1-p_4)^2 = -p_1^2 - p_4^2 + 2p_1\cdot p_4 = m_1^2 + m_4^2 - 2p_1\cdot p_4$$

求和：

$$s + t + u = 3m_1^2 + m_2^2 + m_3^2 + m_4^2 + 2p_1\cdot(p_2 - p_3 - p_4)$$

由动量守恒 $p_2 = p_3 + p_4 - p_1$，得 $p_2 - p_3 - p_4 = -p_1$：

$$s + t + u = 3m_1^2 + m_2^2 + m_3^2 + m_4^2 - 2p_1^2 = 3m_1^2 + m_2^2 + m_3^2 + m_4^2 + 2m_1^2$$

$$= m_1^2 + m_2^2 + m_3^2 + m_4^2 + 2m_1^2 + 2m_1^2$$

> 实际上更简洁的证明是利用 $s + t + u = -(p_1+p_2)^2 - (p_1-p_3)^2 - (p_1-p_4)^2$，结合 $p_1+p_2=p_3+p_4$，直接展开即可得到 $\sum m_i^2$。$\square$

因此，三个 Mandelstam 变量中**只有两个是独立的**——与前面运动学计数的结果一致。

### 3.2 洛伦兹不变性

$s$, $t$, $u$ 都是四维动量的洛伦兹标量——在任何惯性参考系中取值相同。这使得它们成为描述散射运动学的**自然选择**。

### 3.3 交叉对称性

Mandelstam 变量的一个深刻特征是：**同一个散射振幅 $\mathcal{M}(s,t)$ 在不同物理过程中取相同函数形式的不同变量值。**

这是**交叉对称性**（crossing symmetry）的数学表述——将在第五节详细讨论。

## 四、物理意义——质心系中的表达

### 4.1 质心系运动学

在质心系（CM frame）中，$\boldsymbol{p}_1 + \boldsymbol{p}_2 = 0$，即 $\boldsymbol{p}_1 = -\boldsymbol{p}_2 \equiv \boldsymbol{p}$。

总四维动量：

$$P = p_1 + p_2 = (E_1 + E_2,\;\boldsymbol{0})$$

$$\boxed{s = -(p_1 + p_2)^2 = (E_1 + E_2)^2 = E_{\text{cm}}^2}$$

> **$s$ 就是质心系总能量的平方。**

### 4.2 $t$ 的物理意义——动量转移

在质心系中，散射角 $\theta$ 定义为入射方向与出射方向之间的夹角。

设散射前后动量大小分别为 $|\boldsymbol{p}|$ 和 $|\boldsymbol{p}'|$，则：

$$t = -(p_1 - p_3)^2 = -(E_1 - E_3)^2 + |\boldsymbol{p}_1 - \boldsymbol{p}_3|^2$$

对于**弹性散射**（$m_1 = m_3$, $m_2 = m_4$），$E_1 = E_3$，$|\boldsymbol{p}_1| = |\boldsymbol{p}_3|$：

$$\boxed{t = -|\boldsymbol{p}_1 - \boldsymbol{p}_3|^2 = -2|\boldsymbol{p}|^2(1 - \cos\theta)}$$

因此：
- **向前散射**（$\theta = 0$）：$t = 0$
- **向后散射**（$\theta = \pi$）：$t = -4|\boldsymbol{p}|^2$
- 一般情况：$t \leq 0$

> **$t$ 衡量了散射过程中的"动量转移"——$|t|$ 越大，散射越剧烈（偏转角越大）。**

### 4.3 $u$ 的物理意义

类似地，对于弹性散射：

$$\boxed{u = -2|\boldsymbol{p}|^2(1 + \cos\theta) + \text{质量项}}$$

$u$ 与 $t$ 通过 $\cos\theta$ 联系——当 $\theta$ 从 $0$ 变到 $\pi$，$t$ 和 $u$ 在运动学允许区域内互补。

### 4.4 不变量的数值范围

对于弹性散射 $m_1 = m_3 = m$，$m_2 = m_4 = M$：

| 变量 | 范围 | 物理含义 |
|------|------|---------|
| $s$ | $s \geq (m + M)^2$ | 阈值以上 |
| $t$ | $-4p_{\text{cm}}^2 \leq t \leq 0$ | 动量转移（非正） |
| $u$ | $-4p_{\text{cm}}^2 \leq u \leq 0$ | 交叉动量转移（非正） |

## 五、交叉对称性

### 5.1 三种散射道

同一组 Mandelstam 变量描述的不是**一个**物理过程，而是**三个**相关的过程。以 $1 + 2 \to 3 + 4$ 为例：

$$\boxed{
\begin{aligned}
&\boldsymbol{s}\text{-道（s-channel）：}\quad 1 + 2 \to 3 + 4 \\[4pt]
&\boldsymbol{t}\text{-道（t-channel）：}\quad 1 + \bar{3} \to \bar{2} + 4 \\[4pt]
&\boldsymbol{u}\text{-道（u-channel）：}\quad 1 + \bar{4} \to 3 + \bar{2}
\end{aligned}
}$$

其中 $\bar{i}$ 表示粒子 $i$ 的反粒子。

### 5.2 交叉对称性的含义

散射振幅 $\mathcal{M}$ 是 $s$, $t$, $u$ 的函数（对无自旋粒子而言）。**同一个函数 $\mathcal{M}(s,t)$** 在不同物理区域描述不同的散射过程：

| 道 | 物理过程 | 变量角色 |
|----|---------|---------|
| $s$-道 | $1 + 2 \to 3 + 4$ | $s > 0$（正能量），$t < 0$ |
| $t$-道 | $1 + \bar{3} \to \bar{2} + 4$ | $t > 0$（正能量），$s < 0$ |
| $u$-道 | $1 + \bar{4} \to 3 + \bar{2}$ | $u > 0$（正能量），$s < 0$ |

> 想象散射振幅是一个"地形图"，$s$ 和 $t$ 是两个坐标轴。不同的物理过程对应于在这张地形图上的**不同区域**——但地形本身（函数形式）是同一个。

### 5.3 具体例子：$e^-e^+ \to \mu^-\mu^+$

以电子-正电子湮灭产生μ子对为例：

| 道 | 过程 | 中间态 | 物理区域 |
|----|------|--------|---------|
| $s$-道 | $e^-e^+ \to \mu^-\mu^+$ | $\gamma^*/Z^0$ | $s > 4m_\mu^2$，$t < 0$ |
| $t$-道 | $e^-\mu^+ \to e^-\mu^+$ | 光子交换 | $t > 0$，$s < 0$（非物理区） |
| $u$-道 | $e^-\mu^- \to e^-\mu^-$ | 光子交换 | $u > 0$（电子-μ子弹性散射） |

## 六、Mandelstam 变量与费曼图

### 6.1 $s$-道共振

当 $s$ 接近某个中间态粒子的质量平方时，出现 $s$-道共振：

$$\mathcal{M}_s \propto \frac{1}{s - M^2 + iM\Gamma}$$

这就是我们之前讨论的 **Breit-Wigner 共振**——在 $s = M^2$ 处截面出现峰值。

```
s-道费曼图：

  1 ──────●────── 3
          |
          | 中间态（质量 M）
          |
  2 ──────●────── 4

  s = (p₁ + p₂)² = M²（共振条件）
```

### 6.2 $t$-道交换

当 $t$ 接近某个交换粒子的质量平方时，出现 $t$-道极点：

$$\mathcal{M}_t \propto \frac{1}{t - m_{\text{ex}}^2}$$

```
t-道费曼图：

  1 ──────●────── 3
          |
          | 交换粒子（质量 m_ex）
          |
  2 ──────●────── 4

  t = (p₁ - p₃)² = m_ex²（极点条件）
```

| 交换粒子 | 质量 | $t$-道极点 | 物理效应 |
|---------|------|-----------|---------|
| 光子 | $0$ | $t = 0$ | 长程库仑力 |
| $Z^0$ | $91\;\text{GeV}$ | $t = -M_Z^2$ | 短程弱力 |
| $\pi^0$ | $135\;\text{MeV}$ | $t = -m_\pi^2$ | 核力的 Yukawa 势 |

### 6.3 $u$-道交换

类似地，$u$-道极点：

$$\mathcal{M}_u \propto \frac{1}{u - m_{\text{ex}}^2}$$

### 6.4 完整振幅

许多物理过程中，多个道的贡献同时存在。例如 $e^-e^- \to e^-e^-$（Møller 散射）：

$$\mathcal{M} = \mathcal{M}_t + \mathcal{M}_u = \frac{e^2}{t} + \frac{e^2}{u}$$

（树图近似，忽略电子质量）

两个交换光子的贡献分别在 $t = 0$ 和 $u = 0$ 处有极点——对应向前散射和向后散射的库仑增强。

## 七、Mandelstam 表示

### 7.1 色散关系

Mandelstam 最重要的贡献之一是提出了散射振幅的**双重色散关系**（double dispersion relation），也称为 **Mandelstam 表示**：

$$\boxed{\mathcal{M}(s,t) = \frac{1}{\pi}\int\frac{\mathrm{Im}_s\,\mathcal{M}(s',t)}{s' - s}\,ds' + \frac{1}{\pi}\int\frac{\mathrm{Im}_t\,\mathcal{M}(s,t')}{t' - t}\,dt' + \cdots}$$

### 7.2 物理意义

Mandelstam 表示的核心思想是：

> **散射振幅在任意一点的值，完全由其在物理区域的虚部（即总截面，通过光学定理）决定。**

这意味着：
- 散射振幅是 $s$ 和 $t$ 的**解析函数**
- 其奇异性（极点、割线）的位置和类型由物理过程的中间态决定
- 振幅的实部和虚部通过色散关系相互联系

### 7.3 解析结构

在复 $s$ 平面上，散射振幅的奇异性结构为：

| 奇异性类型 | 位置 | 物理来源 |
|-----------|------|---------|
| **极点** | $s = m_{\text{bound}}^2$ | 束缚态 |
| **极点** | $s = (m_1 + m_2)^2$（第二黎曼叶） | 共振态 |
| **正常割线** | $s \geq (m_1 + m_2)^2$ | 两粒子连续谱 |
| **异常割线** | $s \leq (m_3 + m_4 - m_1 - m_2)^2$ 等 | 交叉道阈值 |

> **深刻理解**：Mandelstam 表示将散射振幅的**解析性**（数学性质）与**幺正性**（物理约束）统一起来——这是 S 矩阵理论的基石。

## 八、不同参考系中的 Mandelstam 变量

### 8.1 质心系（CM frame）

$$\boldsymbol{p}_1 = -\boldsymbol{p}_2 = \boldsymbol{p}, \qquad \boldsymbol{p}_3 = -\boldsymbol{p}_4 = \boldsymbol{p}'$$

$$s = (E_1 + E_2)^2$$

对于弹性散射（$|\boldsymbol{p}| = |\boldsymbol{p}'|$）：

$$t = -2|\boldsymbol{p}|^2(1 - \cos\theta_{\text{cm}})$$

$$u = 2(m_1^2 + m_2^2) - s - t$$

### 8.2 实验室系（Lab frame）

粒子 2 静止（$p_2 = (m_2, \boldsymbol{0})$）：

$$s = m_1^2 + m_2^2 + 2m_2 E_1^{\text{lab}}$$

$$t = -2m_2(E_3^{\text{lab}} - E_1^{\text{lab}}) + 2m_2^2 - 2m_2^2\frac{E_3^{\text{lab}}}{m_2}$$

更简洁地，对于弹性散射：

$$t = -2m_2\,T_3^{\text{lab}}$$

其中 $T_3 = E_3 - m_3$ 是末态粒子 3 的动能。

> **实验应用**：在固定靶实验中，测量末态粒子的动能即可直接确定 $t$。

### 8.3 Breit 系（Breit frame）

定义为 $p_1 + p_3 = 0$（粒子 1 和 3 的三动量大小相等、方向相反）。在此参考系中：

$$t = -(p_1 - p_3)^2 = -4|\boldsymbol{p}|^2$$

> Breit 系在深度非弹性散射和形状因子的讨论中特别有用。

## 九、不变质量与 Mandelstam 变量

### 9.1 不变质量的物理意义

$s = -(p_1 + p_2)^2$ 是初态（或末态）总四维动量的模方。在质心系中 $\sqrt{s}$ 就是**总质心能量**。

对于多粒子末态，部分不变质量定义为：

$$s_{ij} = -(p_i + p_j)^2$$

这些部分不变质量在多体衰变的运动学分析（如 **Dalitz 图**）中起着核心作用。

### 9.2 与 Breit-Wigner 的联系

当 $s$ 扫过某个共振态的质量平方 $M^2$ 时，截面出现 Breit-Wigner 峰。Mandelstam 变量 $s$ 正是 Breit-Wigner 分布：

$$|\mathcal{A}(s)|^2 \propto \frac{1}{(s - M^2)^2 + M^2\Gamma^2}$$

中的自变量。

## 十、推广

### 10.1 高能极限

在高能极限 $s \gg m_i^2$ 下，Mandelstam 恒等式简化为：

$$s + t + u \approx 0$$

散射振幅的领头行为通常由**Regge 理论**描述：

$$\mathcal{M}(s,t) \sim s^{\alpha(t)}$$

其中 $\alpha(t)$ 是 **Regge 轨迹**——将粒子的自旋 $J$ 与其质量 $M$ 联系起来：$J = \alpha(M^2)$。

### 10.2 多体过程的推广

对于 $2 \to n$ 散射，独立的 Mandelstam 变量个数为 $3n - 3$。例如 $2 \to 3$ 过程有 $6$ 个独立不变量。

### 10.3 深度非弹性散射

在电子-质子深度非弹性散射 $e^-p \to e^-X$ 中，常用的不变量为：

$$Q^2 = -t = -(p_e - p_e')^2 \quad \text{（虚光子的四维动量平方）}$$

$$\nu = \frac{p \cdot q}{m_p} = \frac{E_e - E_e'}{m_p} \quad \text{（能量转移）}$$

$$x = \frac{Q^2}{2m_p\nu} \quad \text{（Bjorken-}x\text{）}$$

> 这些变量本质上是 Mandelstam 变量在深度非弹性散射运动学中的具体应用。

## 十一、总结

$$\boxed{
\begin{aligned}
&s = -(p_1+p_2)^2 = E_{\text{cm}}^2 \quad\text{（质心能量平方）} \\[4pt]
&t = -(p_1-p_3)^2 \leq 0 \quad\text{（动量转移平方）} \\[4pt]
&u = -(p_1-p_4)^2 \leq 0 \quad\text{（交叉动量转移平方）} \\[6pt]
&s + t + u = \sum_i m_i^2 \quad\text{（Mandelstam 恒等式）}
\end{aligned}
}$$

| 概念 | 内容 |
|------|------|
| **定义** | 由四维动量构造的三个洛伦兹不变量 |
| **独立性** | 三个变量中只有两个独立 |
| **物理意义** | $s$：能量；$t$：动量转移；$u$：交叉动量转移 |
| **交叉对称性** | 同一振幅函数描述三个相关物理过程 |
| **费曼图** | $s$-道、$t$-道、$u$-道对应不同的中间态交换 |
| **解析性** | Mandelstam 表示将振幅的解析结构与物理截面联系 |
| **实验应用** | 直接对应可观测量（能量、散射角、不变质量） |

Mandelstam 变量是粒子物理散射理论的**运动学语言**——它将复杂的四维相对论运动学压缩为最少的洛伦兹不变量，使得散射振幅的对称性、解析性和物理意义变得透明。
