
自然单位制是理论物理学中一种极简且优雅的单位制，其核心思想是：**将自然界的基本物理常数设为1（无量纲化）**。

这样做可以极大地简化物理定律的数学形式，揭示物理量之间更深层次的本质联系。

---

### 核心思想与目的

1.  **简化公式**：许多物理公式中都包含像光速 $c$、约化普朗克常数 $\hbar$、玻尔兹曼常数 $k_B$、万有引力常数 $G$ 这样的常数。通过设定它们为1，可以消除这些“累赘”，让公式变得极其简洁。
2.  **揭示本质**：当这些常数消失后，物理量之间的关系变得更加直接。例如，能量、质量、频率和温度在本质上是同一回事，只是通过不同的常数进行换算。自然单位制揭示了这种统一性。
3.  **聚焦物理**：在计算和推导时，无需反复书写和处理这些常数，可以更专注于物理思想本身。

---

*   **设定为1的常数**：
    *   光速： $c = 1$
    *   约化普朗克常数： $\hbar = 1$
    *   真空介电常数（有时）： $\epsilon_0 = 1$ 或 $4\pi\epsilon_0 = 1$

*   **由此带来的简化**：
    *   **能量、质量、动量**的量纲相同，单位都是 eV（电子伏特）。
        *   $E = mc^2$ 变为 $E = m$。
        *   $E = pc$ 变为 $E = p$。
    *   **长度和时间**的量纲相同，都是能量的倒数。
        *   由于 $c = 1$，所以长度和时间可以互换： $x = ct$ 变为 $x = t$。
        *   具体换算： $1 \text{ eV}^{-1} \approx 1.97 \times 10^{-7} \text{ m} \approx 6.58 \times 10^{-16} \text{ s}$。
    *   **温度**与能量直接关联。
        *   由于 $k_B = 1$，温度 T 的单位也是 eV。
        *   具体换算： $1 \text{ eV} \approx 11604 \text{ K}$。


# 自然单位制（$\hbar = c = k_B = 1$）中的量纲分析

在自然单位制中，由于 $\hbar = c = k_B = 1$，所有物理量的量纲都可以用单一质量量纲 $M$ 来表达：

$$L = T = \Theta^{-1} = M^{-1}$$

|      量纲      | 物理量                                                                                                                                                   |
| :----------: | :---------------------------------------------------------------------------------------------------------------------------------------------------- |
| $M^{0}$（无量纲） | 速度、角速度、电荷（Gauss制）、精细结构常数 $\alpha$、耦合常数（Yukawa 耦合 $g$，规范耦合 $g_1,g_2,g_3$）、角度、立体角、相位、螺旋度、粒子数、重子数 $B$、轻子数 $L$、熵 $S$、角动量 $J$、作用量 $S$、$\beta = 1/T$、$GT^2$ |
|   $M^{-2}$   | 截面 $\sigma$、Fermi 常数 $G_F$、Newton 引力常数 $G$                                                                                                            |
|   $M^{-1}$   | 长度、时间、Compton 泓长 $\lambda = 1/m$、Coulomb 势 $V = \alpha/r$、汤川势 $e^{-mr}/r$、衰变宽度的倒数 $\Gamma^{-1}$（寿命）、逆温度 $\beta = 1/T$                                 |
|   $M^{1}$    | 质量、能量、动量、温度、频率 $\omega$、化学势 $\mu$、标量场 $\phi$（4 维）、规范势 $A_\mu$、Higgs 场真空期望值 $v$、Fermi 能 $E_F$、Debye 屏蔽质量 $m_D$、加速度 $a$                                 |
|   $M^{2}$    | 力 $F = dp/dt$、功率 $P = dE/dt$、表面张力、规范耦合的跑动 $\Lambda_{\text{QCD}}$、质量平方 $m^2$（如 Higgs 质量项 $\mu^2\|\phi\|^2$ 中 $\mu^2$）                                  |
|   $M^{3}$    | Riemann 曲率张量 $R^\rho{}_{\sigma\mu\nu}$、电磁场张量 $F_{\mu\nu}$、电场 $\mathbf{E}$、磁场 $\mathbf{B}$、旋量场 $\psi$                                                  |
|   $M^{4}$    | Lagrangian 密度 $\mathcal{L}$、Hamiltonian 密度 $\mathcal{H}$、能量密度 $\varepsilon$、压强 $p$、应力-能量张量 $T_{\mu\nu}$、Ricci 标量曲率 $R$                                |

### 核心规律

> **一个物理量在 $d$ 维时空中的质量量纲 = $d$ − 该量纲在作用量 $S = \int d^dx\, \mathcal{L}$ 中的"权重"。**

更直接地说，在 4 维中：
- $[\mathcal{L}] = M^4$ 是基准
- 场的量纲由动能项 $\sim (\partial\phi)^2$ 固定
- 耦合常数的量纲 = $4 - n \cdot [\text{场}] - [\text{导数}]$，其中 $n$ 是场的个数

这也是为什么 4 维中**只有 [[Yukawa 耦合]]和[[规范耦合]]是无量纲的**，而**[[引力耦合]]** $G$ 有**负量纲 $M^{-2}$**（不可重整化）的深层原因。

### 如何从自然单位恢复到国际单位制（SI）

当你需要得到有量纲的数值结果时，可以通过 **量纲分析** 来恢复。方法是：检查你的最终表达式，补上适当的 $c$、$\hbar$、$k_B$ 的幂次，使得整个表达式的量纲正确。

**举个例子**：
在自然单位制中，一个粒子的康普顿波长写作 $\lambda = 1/m$。
我们知道波长的量纲是 [L]，质量的量纲是 [M]。在SI单位制中，[L] 和 [M] 不能直接相等。
我们需要组合常数来得到长度的量纲： $c$ 的量纲是 [L][T]⁻¹， $\hbar$ 的量纲是 [M][L]²[T]⁻¹。
可以验证，$\frac{\hbar}{mc}$ 的量纲是 [L]。
因此，康普顿波长的完整表达式是 $\lambda = \frac{\hbar}{mc}$。

### 总结

| 特征 | 自然单位制 | 国际单位制（SI） |
| :--- | :--- | :--- |
| **目的** | 揭示物理本质，简化理论推导 | 便于实验测量、工程应用和日常生活 |
| **基本常数** | $c, \hbar, k_B, G$ 等设为1 | 基本单位独立定义（米、千克、秒等） |
| **优点** | 公式简洁，物理关系清晰 | 数值具体，便于实际操作和交流 |
| **缺点** | 数值不直观，需换算 | 公式冗长，掩盖了某些内在联系 |
| **使用场景** | 理论物理研究、学术论文、思想实验 | 实验物理、工程技术、日常生活 |

**简单来说，自然单位制就像是物理学的“行话”或“简写”**。在理论物理学家之间交流时，使用它可以高效地沟通核心思想。而当需要把结果告诉实验学家或工程师时，再把它“翻译”回通用的国际单位制。它是理论物理学家追求简洁与普适之美的一种体现。
