## 一、总览：四种基本相互作用

自然界中已知的所有粒子间相互作用，都可以归结为**四种基本力**：

| 相互作用 | 作用对象 | 媒介粒子（规范玻色子） | 相对强度 | 力程 | 理论框架 |
|---------|---------|----------------------|---------|------|---------|
| **强相互作用** | 夸克（色荷） | 胶子 $g$（8种） | $1$ | $\sim 10^{-15}$ m（禁闭） | QCD |
| **电磁相互作用** | 带电粒子 | 光子 $\gamma$ | $\sim 10^{-2}$ | $\infty$ | QED |
| **弱相互作用** | 所有费米子 | $W^\pm$, $Z^0$ | $\sim 10^{-6}$ | $\sim 10^{-18}$ m | 电弱理论 |
| **引力相互作用** | 所有有质量/能量的物体 | 引力子 $G$（假设） | $\sim 10^{-39}$ | $\infty$ | 广义相对论（经典） |

> 粒子之间的每一次相互作用，都可以看作是在交换某种"信使粒子"（规范玻色子）。两个电子之间的电磁排斥，本质上是它们不断交换虚光子的过程。力的"强度"取决于交换的效率，力程取决于信使粒子的质量——无质量粒子（如光子）传递长程力，有质量粒子（如 $W^\pm$, $Z^0$）传递短程力。

## 二、相互作用的理论基础：规范对称性与 Yang-Mills 理论

### 1. 规范原理（Gauge Principle）

现代粒子物理中，所有基本相互作用都源于**局域规范对称性**（local gauge symmetry）。

**核心思想**：如果要求自由粒子的拉格朗日量在**局域**（依赖于时空点的）某种对称变换下不变，就必须引入新的场——**规范场**——来"补偿"局域变换带来的额外项。这些规范场正是相互作用的媒介。

### 2. 从全局对称性到[[局域对称性]]

以带电标量场 $\phi$ 为例。

**全局 $U(1)$ 对称性**：

$$\phi \to e^{i\alpha}\phi \qquad (\alpha = \text{常数})$$

拉格朗日量 $\mathcal{L} = |\partial_\mu\phi|^2 - m^2|\phi|^2$ 在此变换下不变。由 Noether 定理，这给出守恒的电荷流。

**局域 $U(1)$ 对称性**：

$$\phi \to e^{i\alpha(x)}\phi \qquad (\alpha = \alpha(x)\;\text{依赖于时空点})$$

此时 $\partial_\mu\phi$ 的变换多出一项 $i(\partial_\mu\alpha)\phi$，拉格朗日量**不再不变**。

**解决方案**：引入规范场 $A_\mu$，将普通导数替换为**协变导数**：

$$\boxed{\partial_\mu \to D_\mu = \partial_\mu + ieA_\mu}$$

并要求 $A_\mu$ 在规范变换下：

$$A_\mu \to A_\mu - \frac{1}{e}\partial_\mu\alpha$$

这样 $D_\mu\phi$ 与 $\phi$ 以相同方式变换，$\mathcal{L} = |D_\mu\phi|^2 - m^2|\phi|^2$ 保持不变。

规范场 $A_\mu$ 就是**光子场**，$e$ 是**耦合常数**（电荷），由此产生的相互作用就是**电磁相互作用**。

### 3. 非阿贝尔规范理论（Yang-Mills 理论）

当规范群是**非阿贝尔**（不可交换）群时，规范场自身也携带荷，可以自相互作用。

设规范群为 $G$，生成元为 $T^a$（$a = 1, \ldots, \dim G$），结构常数 $f^{abc}$：

$$[T^a, T^b] = if^{abc}T^c$$

协变导数：

$$D_\mu = \partial_\mu - igA_\mu^a T^a$$

规范场的场强张量：

$$\boxed{F_{\mu\nu}^a = \partial_\mu A_\nu^a - \partial_\nu A_\mu^a + gf^{abc}A_\mu^b A_\nu^c}$$

Yang-Mills 拉格朗日量：

$$\boxed{\mathcal{L}_{\text{YM}} = -\frac{1}{4}F_{\mu\nu}^a F^{a\mu\nu}}$$

关键特征：$F_{\mu\nu}^a$ 中的 $gf^{abc}A_\mu^b A_\nu^c$ 项导致了**规范场的自耦合**——规范玻色子之间可以直接相互作用。这在阿贝尔理论（QED）中不存在（光子不带电荷，光子之间不直接耦合）。

### 4. 标准模型的规范群

$$\boxed{G_{\text{SM}} = SU(3)_C \times SU(2)_L \times U(1)_Y}$$

| 规范群 | 相互作用 | 规范玻色子 | 耦合常数 | 作用对象 |
|--------|---------|-----------|---------|---------|
| $SU(3)_C$ | 强 | 8 个胶子 $g^a$ | $g_s$（或 $\alpha_s = g_s^2/4\pi$） | 夸克（色荷） |
| $SU(2)_L$ | 弱同位旋 | $W_1, W_2, W_3$ | $g$ | 左手费米子 |
| $U(1)_Y$ | 弱超荷 | $B$ | $g'$ | 所有粒子（按超荷） |

电弱对称性自发破缺后：

$$SU(2)_L \times U(1)_Y \xrightarrow{\text{Higgs}} U(1)_{\text{em}}$$

物理的规范玻色子变为：
- $\gamma = \sin\theta_W\,W_3 + \cos\theta_W\,B$（光子，无质量）
- $Z = \cos\theta_W\,W_3 - \sin\theta_W\,B$（$Z$ 玻色子，有质量）
- $W^\pm = (W_1 \mp iW_2)/\sqrt{2}$（$W$ 玻色子，有质量）

其中 $\theta_W$ 是 **Weinberg 角**（$\sin^2\theta_W \approx 0.231$）。

## 三、相互作用的耦合常数与强度

### 1. 耦合常数的定义

每种相互作用由一个**耦合常数** $g$（或等价地 $\alpha = g^2/4\pi$）表征，它决定了相互作用的"强度"。

| 相互作用 | 耦合常数 | 典型值 | 能标依赖 |
|---------|---------|--------|---------|
| 电磁 | $\alpha = e^2/4\pi$ | $\approx 1/137$（低能） | **增大**（对数增长） |
| 强 | $\alpha_s = g_s^2/4\pi$ | $\approx 0.12$（$M_Z$ 能标） | **减小**（渐近自由） |
| 弱 | $\alpha_W = g^2/4\pi$ | $\approx 1/30$（$M_W$ 能标） | 类似 QED |
| 引力 | $\alpha_G = G_N m^2/4\pi$ | $\sim 10^{-39}$（质子质量） | — |

### 2. 跑动耦合常数（Running Coupling Constants）

耦合常数不是真正的"常数"——它随相互作用的**能量标度** $\mu$ 变化。这是量子场论中**重整化**的深刻结果。

**重整化群方程**（RGE）：

$$\boxed{\mu\frac{d\alpha}{d\mu} = \beta(\alpha) = -b_0\,\alpha^2 - b_1\,\alpha^3 - \cdots}$$

其中 $\beta$ 函数的系数取决于理论的内容（费米子和玻色子的数目）。

**QED**（$\beta > 0$）：

$$b_0^{\text{QED}} = -\frac{1}{3\pi}\sum_f Q_f^2 < 0 \quad \Rightarrow \quad \beta_{\text{QED}} > 0$$

$\alpha$ 随能量**增大**——在高能（短距离）下电磁相互作用变强。

**QCD**（$\beta < 0$）：

$$b_0^{\text{QCD}} = \frac{1}{12\pi}(11N_c - 2N_f) = \frac{1}{12\pi}(33 - 2N_f) > 0 \quad (N_f \leq 16)$$

$$\beta_{\text{QCD}} = -b_0\alpha_s^2 < 0$$

$\alpha_s$ 随能量**减小**——**渐近自由**（asymptotic freedom），在高能下夸克行为近似自由粒子。

```
α(μ)
  ↑
  │  QED ╱
  │     ╱        α_s (QCD)
  │    ╱       ╲
  │   ╱         ╲
  │  ╱           ╲
  │ ╱    α_EM      ╲
  │╱                 ╲
  ┼───────────────────────→ log μ
     低能              高能
```

### 3. 统一的图景：大统一理论（GUT）

三种规范耦合常数在高能标下趋向汇聚（在最小超对称标准模型中尤为精确）：

$$\alpha_1(\mu) \approx \alpha_2(\mu) \approx \alpha_3(\mu) \quad \text{在} \quad \mu \sim 10^{15}\text{--}10^{16}\;\text{GeV}$$

这暗示在极高能量下，三种力可能统一为一种——**大统一理论**（GUT），规范群可能为 $SU(5)$、$SO(10)$ 等。

## 四、电磁相互作用（QED）

### 1. 拉格朗日量

$$\boxed{\m
athcal{L}_{\text{QED}} = \bar{\psi}(i\slashed{D} - m)\psi - \frac{1}{4}F_{\mu\nu}F^{\mu\nu} = \bar{\psi}(i\slashed{\partial} - m)\psi - e\bar{\psi}\gamma^\mu\psi A_\mu - \frac{1}{4}F_{\mu\nu}F^{\mu\nu}}$$

### 2. 费曼规则

**传播子**（光子，Feynman 规范）：

$$\widetilde{\gamma}: \quad \frac{-i\eta_{\mu\nu}}{q^2 + i\epsilon}$$

**传播子**（费米子）：

$$\widetilde{f}: \quad \frac{i(\slashed{p} + m)}{p^2 - m^2 + i\epsilon}$$

**顶点**：

$$e^-e^-\gamma \;\text{顶点}: \quad -ie\gamma^\mu$$

每个顶点贡献一个因子 $-ie\gamma^\mu$，并赋予费米子线一个电荷因子。

### 3. 典型过程与截面

**$e^-e^- \to e^-$ 散射（Møller 散射）**：

$$\frac{d\sigma}{d\Omega} = \frac{\alpha^2}{2s}\left[\frac{s^2+u^2}{t^2} + \frac{s^2+t^2}{u^2} + \frac{2s^2}{tu}\right]$$

**$e^+e^- \to \mu^+\mu^-$ 散射**：

$$\frac{d\sigma}{d\Omega} = \frac{\alpha^2}{4s}(1+\cos^2\theta)$$

**Compton 散射** $\gamma e^- \to \gamma e^-$（Klein-Nishina 公式）：

$$\frac{d\sigma}{d\Omega} = \frac{\alpha^2}{2m^2}\left(\frac{\omega'}{\omega}\right)^2\left[\frac{\omega'}{\omega} + \frac{\omega}{\omega'} - \sin^2\theta\right]$$

### 4. QED 的精密预言

QED 的高阶计算代表了理论物理精度的巅峰：

| 可观测量 | 理论精度 | 实验精度 | 吻合程度 |
|---------|---------|---------|---------|
| 电子 $g-2$ | $10^{-13}$ | $10^{-13}$ | 完美 |
| Lamb 位移 | $\sim 10^{-6}$ | $\sim 10^{-6}$ | 极好 |
| 缪子 $g-2$ | $10^{-10}$ | $10^{-10}$ | 存在争议 |

## 五、强相互作用（QCD）

### 1. 拉格朗日量

$$\boxed{\mathcal{L}_{\text{QCD}} = \bar{q}_i(i\slashed{D}_{ij} - m_i\delta_{ij})q_j - \frac{1}{4}G_{\mu\nu}^a G^{a\mu\nu}}$$

其中：
- $q_i$：夸克场（$i = 1,2,3$ 为色指标，$f = u,d,s,c,b,t$ 为味指标）
- $D_\mu = \partial_\mu - ig_s A_\mu^a T^a$：色协变导数
- $G_{\mu\nu}^a$：胶子场强张量
- $T^a = \lambda^a/2$：$SU(3)$ 的生成元（$\lambda^a$ 为 Gell-Mann 矩阵）

### 2. 费曼规则

**胶子传播子**（Feynman 规范）：

$$\frac{-i\delta^{ab}\eta_{\mu\nu}}{q^2 + i\epsilon}$$

**夸克-胶子顶点**：

$$-ig_s\gamma^\mu T^a$$

**三胶子顶点**（非阿贝尔理论独有）：

$$-g_s f^{abc}\left[\eta^{\mu\nu}(k_1-k_2)^\rho + \eta^{\nu\rho}(k_2-k_3)^\mu + \eta^{\rho\mu}(k_3-k_1)^\nu\right]$$

**四胶子顶点**（非阿贝尔理论独有）：

$$-ig_s^2\left[f^{abe}f^{cde}(\eta^{\mu\rho}\eta^{\nu\sigma}-\eta^{\mu\sigma}\eta^{\nu\rho}) + \text{置换}\right]$$

### 3. QCD 的两大特征

#### (a) 渐近自由（Asymptotic Freedom）

在高能（短距离）下，$\alpha_s \to 0$，夸克行为近似自由：

$$\alpha_s(Q^2) = \frac{12\pi}{(33-2N_f)\ln(Q^2/\Lambda_{\text{QCD}}^2)}$$

其中 $\Lambda_{\text{QCD}} \approx 200$ MeV 是 QCD 的特征能标。

在 LHC（$Q \sim 1$ TeV）：$\alpha_s \approx 0.1$，微扰论适用。

#### (b) 色禁闭（Color Confinement）

在低能（长距离）下，$\alpha_s$ 增大到微扰论失效。实验上从未观测到自由的夸克或胶子——它们总是被"禁闭"在色单态的强子内部。

夸克之间的势能近似为**线性势**：

$$V(r) \approx -\frac{4}{3}\frac{\alpha_s}{r} + \sigma r \qquad (r \to \infty)$$

其中 $\sigma \approx 1$ GeV/fm 是**弦张力**。线性势意味着分离两个夸克需要的能量随距离线性增长，最终足以从真空中产生新的 $q\bar{q}$ 对——这就是**弦断裂**（string breaking）。

### 4. 强子结构

由于色禁闭，物理的强子是**色单态**：

| 强子类型 | 夸克组成 | 色结构 | 例子 |
|---------|---------|--------|------|
| 介子 | $q\bar{q}$ | $\frac{1}{\sqrt{3}}(r\bar{r}+g\bar{g}+b\bar{b})$ | $\pi$, $K$, $\rho$, $J/\psi$ |
| 重子 | $qqq$ | $\frac{1}{\sqrt{6}}\epsilon^{ijk}q_iq_jq_k$ | $p$, $n$, $\Lambda$, $\Delta$ |
| 奇异强子 | $qq\bar{q}\bar{q}$, $qqqq\bar{q}$ 等 | 色单态 | 四夸克态、五夸克态 |

## 六、弱相互作用

### 1. 基本特征

弱相互作用有几个独一无二的特征：

| 特征 | 内容 |
|------|------|
| **手征性** | 只与左手费米子（和右手反费米子）耦合 |
| **味改变** | 可以改变夸克和轻子的味（flavor） |
| **宇称不守恒** | 最大程度地违反宇称对称性 |
| **CP 破坏** | 通过 CKM 矩阵的复相位 |
| **有质量媒介子** | $W^\pm$（$\sim 80.4$ GeV），$Z^0$（$\sim 91.2$ GeV） |
| **短程力** | 力程 $\sim 1/M_W \sim 10^{-18}$ m |

### 2. 弱相互作用的两种类型

#### (a) 带电流（Charged Current, CC）——$W^\pm$ 交换

$$\mathcal{L}_{\text{CC}} = -\frac{g}{\sqrt{2}}\left(\bar{\psi}_i\gamma^\mu P_L \psi_j\,W_\mu^+ + \text{h.c.}\right)$$

其中 $P_L = \frac{1-\gamma^5}{2}$ 是左手投影算符。

**味改变**通过 **CKM 矩阵**（夸克）或 **PMNS 矩阵**（轻子）实现：

$$\mathcal{L}_{\text{CC}}^{(\text{夸克})} = -\frac{g}{\sqrt{2}}\bar{u}_i\gamma^\mu P_L\,V_{ij}^{\text{CKM}}\,d_j\,W_\mu^+ + \text{h.c.}$$

$$V^{\text{CKM}} = \begin{pmatrix} V_{ud} & V_{us} & V_{ub} \\ V_{cd} & V_{cs} & V_{cb} \\ V_{td} & V_{ts} & V_{tb} \end{pmatrix} \approx \begin{pmatrix} 0.974 & 0.225 & 0.004 \\ 0.225 & 0.973 & 0.041 \\ 0.009 & 0.040 & 0.999 \end{pmatrix}$$

典型过程：
- $\beta$ 衰变：$n \to p\,e^-\bar{\nu}_e$（$d \to u\,W^-$，$W^- \to e^-\bar{\nu}_e$）
- $\pi^+ \to \mu^+\nu_\mu$（$u\bar{d} \to W^+ \to \mu^+\nu_\mu$）
- 中微子散射：$\nu_\mu + n \to \mu^- + p$

#### (b) 中性流（Neutral Current, NC）——$Z^0$ 交换

$$\mathcal{L}_{\text{NC}} = -\frac{g}{2\cos\theta_W}\bar{\psi}\gamma^\mu(g_V - g_A\gamma^5)\psi\,Z_\mu$$

其中：
$$g_V = T_3 - 2Q\sin^2\theta_W, \qquad g_A = T_3$$

$T_3$ 是弱同位旋第三分量，$Q$ 是电荷。

典型过程：
- $\nu_\mu + e^- \to \nu_\mu + e^-$（中微子-电子弹性散射）
- $e^+e^- \to f\bar{f}$ 中的 $Z$ 玻色子交换

### 3. 弱相互作用的有效理论

在低能（$E \ll M_W$）下，$W$ 和 $Z$ 的传播子可以被"积掉"，得到**四费米子有效相互作用**：

$$\mathcal{L}_{\text{eff}} = -\frac{G_F}{\sqrt{2}}J_\mu^+ J^{-\mu}$$

其中 Fermi 常数：

$$\frac{G_F}{\sqrt{2}} = \frac{g^2}{8M_W^2} \approx 1.166 \times 10^{-5}\;\text{GeV}^{-2}$$

> 在低能下，你"看不到"$W$ 和 $Z$ 玻色子——它们太重了，只能作为虚粒子短暂存在。有效理论将这种虚交换约化为费米子之间的"点接触"相互作用。这就是费米在 1933 年提出的 $\beta$ 衰变理论的现代理解。

### 4. 电弱统一

电磁力和弱力在能量高于 $\sim 100$ GeV 时统一为**电弱相互作用**。对称性自发破缺（Higgs 机制）将 $W_3$ 和 $B$ 混合为物理的 $\gamma$ 和 $Z$：

$$M_W = \frac{gv}{2} \approx 80.4\;\text{GeV}$$
$$M_Z = \frac{M_W}{\cos\theta_W} \approx 91.2\;\text{GeV}$$
$$M_\gamma = 0$$

其中 $v \approx 246$ GeV 是 Higgs 场的真空期望值。

## 七、引力相互作用

### 1. 与量子力学的冲突

引力是唯一尚未成功量子化的基本力。广义相对论是一个经典的几何理论：

$$G_{\mu\nu} = 8\pi G_N T_{\mu\nu}$$

试图将其量子化导致**不可重整**的紫外发散——需要无穷多个反项。

### 2. 引力的极其微弱

[[引力耦合]]强度（以两个质子为例）：

$$\frac{F_{\text{引力}}}{F_{\text{电磁}}} = \frac{G_N m_p^2}{e^2/(4\pi)} \sim 10^{-36}$$

> 两个质子之间的引力比电磁力弱 $10^{36}$ 倍。这就是为什么在粒子物理中引力完全可以忽略——只有在黑洞附近或宇宙极早期，量子引力效应才可能变得重要。

### 3. 量子引力的候选理论

| 方案 | 核心思想 |
|------|---------|
| 弦理论 | 基本对象是一维弦，引力子是闭弦的振动模式 |
| 圈量子引力 | 时空本身被量子化，具有离散的几何结构 |
| 渐近安全引力 | 引力在紫外极限下存在非高斯不动点 |

## 八、Higgs 机制与质量的起源

### 1. 质量问题

规范对称性禁止规范玻色子和费米子直接拥有质量项（质量项 $\bar{\psi}\psi$ 不满足手征对称性；$M^2 A_\mu A^\mu$ 不满足规范不变性）。但实验中 $W$, $Z$, 夸克和轻子都有质量。

### 2. Higgs 机制

引入复标量二重态 $\Phi$（Higgs 场），其势能为：

$$V(\Phi) = \mu^2|\Phi|^2 + \lambda|\Phi|^4 \qquad (\mu^2 < 0, \;\lambda > 0)$$

势能的最小值不在 $\Phi = 0$，而在：

$$\langle\Phi\rangle = \frac{1}{\sqrt{2}}\begin{pmatrix} 0 \\ v \end{pmatrix}, \qquad v = \sqrt{-\mu^2/\lambda} \approx 246\;\text{GeV}$$

这个非零真空期望值**自发破缺**了 $SU(2)_L \times U(1)_Y$ 对称性，但保留了 $U(1)_{\text{em}}$。

**规范玻色子获得质量**：

$$M_W = \frac{gv}{2} \approx 80.4\;\text{GeV}, \qquad M_Z = \frac{v\sqrt{g^2+g'^2}}{2} \approx 91.2\;\text{GeV}, \qquad M_\gamma = 0$$

**费米子获得质量**（[[Yukawa 耦合]]）：

$$\mathcal{L}_{\text{Yukawa}} = -y_f\bar{\psi}_L\Phi\psi_R + \text{h.c.} \quad \xrightarrow{\langle\Phi\rangle = v/\sqrt{2}} \quad m_f = \frac{y_f v}{\sqrt{2}}$$

**Higgs 玻色子**：对称性破缺后，Higgs 场的 4 个自由度中 3 个被 $W^\pm$ 和 $Z$ "吃掉"（成为它们的纵向极化分量），剩余 1 个成为物理的 Higgs 标量玻色子，质量 $M_H \approx 125$ GeV（2012 年 LHC 发现）。

## 九、相互作用顶点的系统分类

### 1. QED 顶点

| 顶点 | 费曼规则 | 耦合常数 |
|------|---------|---------|
| $f\bar{f}\gamma$ | $-ieQ_f\gamma^\mu$ | $eQ_f$ |

只有这一种基本顶点（光子不自耦合）。

### 2. QCD 顶点

| 顶点 | 费曼规则 | 耦合常数 |
|------|---------|---------|
| $q\bar{q}g$ | $-ig_s\gamma^\mu T^a$ | $g_s$ |
| $ggg$ | $-g_sf^{abc}[\cdots]$ | $g_s$ |
| $gggg$ | $-ig_s^2[\cdots]$ | $g_s^2$ |

胶子的自耦合（三胶子和四胶子顶点）是 QCD 独有的特征。

### 3. 电弱顶点

| 顶点 | 费曼规则 | 耦合常数 |
|------|---------|---------|
| $f\bar{f}\gamma$ | $-ieQ_f\gamma^\mu$ | $eQ_f$ |
| $f\bar{f}Z$ | $-ig_Z\gamma^\mu(g_V^f - g_A^f\gamma^5)$ | $g/(2\cos\theta_W)$ |
| $f_u\bar{f}_d W^+$ | $-ig\gamma^\mu P_L V_{ij}/\sqrt{2}$ | $g/\sqrt{2}$ |
| $WW\gamma$ | $-ie[\eta^{\mu\nu}(k_1-k_2)^\rho + \cdots]$ | $e$ |
| $WWZ$ | $-ig\cos\theta_W[\cdots]$ | $g\cos\theta_W$ |
| $WWH$ | $igM_W\eta^{\mu\nu}$ | $gM_W$ |
| $ZZH$ | $igM_Z/\cos^2\theta_W\cdot\eta^{\mu\nu}$ | $gM_Z/\cos^2\theta_W$ |
| $f\bar{f}H$ | $-im_f/v$ | $m_f/v$ |

### 4. 顶点的通用结构

每个基本顶点可以分解为：

$$\text{顶点因子} = (\text{规范耦合}) \times (\text{Lorentz 结构}) \times (\text{味/色结构})$$

## 十、相互作用的层级结构与有效场论

### 1. 能标层级

```
能量尺度 (GeV)
  ↑
  │  10¹⁹  ──── Planck 尺度（量子引力）
  │  10¹⁶  ──── GUT 尺度
  │  10³   ──── 电弱统一 (~100 GeV)
  │  10⁰   ──── QCD 尺度 ΛQCD (~0.2 GeV)
  │  10⁻²  ──── 原子核
  │  10⁻³  ──── 原子
  └──────────→
```

### 2. 有效场论（EFT）的思想

在给定的能量标度 $\mu$ 下，只保留**相关的自由度**，将更重的自由度"积掉"（integrate out），得到低能有效理论：

| 能标 | 完整理论 | 有效理论 |
|------|---------|---------|
| $\mu \gg M_W$ | 电弱标准模型 | — |
| $\mu \ll M_W$ | 标准模型 | Fermi 四费米子理论 |
| $\mu \gg \Lambda_{\text{QCD}}$ | QCD | 微扰 QCD（部分子模型） |
| $\mu \sim \Lambda_{\text{QCD}}$ | QCD | 手征微扰论（$\chi$PT） |
| $\mu \ll m_\pi$ | QCD | 核力的有效理论 |

> 有效场论就像物理学中的"分辨率调节"——当你用低分辨率（低能量）看世界时，你看不到细节，但可以用简化的理论来描述宏观行为。费米的 $\beta$ 衰变理论就是标准模型在低能下的有效理论。

## 十一、散射振幅的统一描述

### 1. 费曼图的系统规则

所有基本相互作用的散射振幅都可以通过费曼图系统地计算：

**步骤**：
1. 画出所有可能的费曼图（按圈数/耦合常数幂次排列）
2. 对每个图，写出传播子和顶点因子
3. 对内部动量积分（圈图），对外部动量施加守恒约束
4. 对费米子线应用统计规则（反对称化）

### 2. 微扰展开

散射振幅按耦合常数的幂次展开：

$$\mathcal{M} = \mathcal{M}^{(0)} + \mathcal{M}^{(1)} + \mathcal{M}^{(2)} + \cdots$$

| 阶数 | 费曼图 | 贡献 |
|------|--------|------|
| 树图（leading order） | 无圈 | $\mathcal{M}^{(0)} \sim g^n$ |
| 单圈（NLO） | 1 个圈 | $\mathcal{M}^{(1)} \sim g^{n+2}$ |
| 双圈（NNLO） | 2 个圈 | $\mathcal{M}^{(2)} \sim g^{n+4}$ |

### 3. 不同相互作用的微扰性

| 相互作用 | 微扰论是否适用 | 原因 |
|---------|--------------|------|
| QED（高能） | **是** | $\alpha \approx 1/137 \ll 1$ |
| QCD（高能） | **是** | $\alpha_s(Q^2 \gg \Lambda^2) \ll 1$（渐近自由） |
| QCD（低能） | **否** | $\alpha_s(Q^2 \sim \Lambda^2) \sim 1$，需非微扰方法 |
| 弱（$E \ll M_W$） | **是** | 有效耦合 $G_F E^2 \ll 1$ |
| 引力 | **否**（通常） | $\alpha_G \sim E^2/M_{\text{Pl}}^2$，在 $E \to M_{\text{Pl}}$ 时发散 |

## 十二、相互作用的守恒律与对称性

### 1. Noether 定理的层次

| 对称性 | 守恒量 | 相关相互作用 |
|--------|--------|-------------|
| 时间平移 | 能量 | 所有 |
| 空间平移 | 动量 | 所有 |
| 空间旋转 | 角动量 | 所有 |
| $U(1)_{\text{em}}$ | 电荷 | 电磁 |
| $SU(3)_C$ | 色荷 | 强 |
| $U(1)_B$ | 重子数 | 所有（精确） |
| $U(1)_L$ | 轻子数 | 所有（精确，中微子振荡下需推广为族轻子数） |
| $C$（电荷共轭） | $C$ 宇称 | 强、电磁（守恒）；弱（不守恒） |
| $P$（宇称） | 空间反射 | 强、电磁（守恒）；弱（最大破坏） |
| $T$（时间反演） | 时间反演 | 强、电磁（守恒）；弱（通过 CPT 定理间接约束） |
| $CP$ | $CP$ 宇称 | 强、电磁（守恒）；弱（微小破坏） |

### 2. $C$, $P$, $T$ 在各相互作用中的行为

| 对称性 | 强 | 电磁 | 弱 |
|--------|-----|------|-----|
| $C$ | ✓ | ✓ | ✗ |
| $P$ | ✓ | ✓ | ✗（最大破坏） |
| $T$ | ✓ | ✓ | ✗（间接） |
| $CP$ | ✓ | ✓ | ✗（K 介子、B 介子中发现） |
| $CPT$ | ✓ | ✓ | ✓（所有相互作用中精确守恒） |

### 3. CPT 定理

**CPT 定理**：任何满足洛伦兹不变性、局域性和正定能量的量子场论中，$CPT$ 联合变换精确守恒。

推论：
- 粒子与反粒子的质量严格相等：$m_p = m_{\bar{p}}$（实验验证到 $10^{-12}$ 精度）
- 粒子与反粒子的寿命严格相等
- 粒子与反粒子的磁矩大小严格相等

## 十四、超越标准模型的新相互作用

### 1. 标准模型的不足

| 问题 | 说明 |
|------|------|
| 引力未纳入 | 量子引力问题 |
| 暗物质 | 标准模型中无暗物质候选者 |

| 等级问题 | Higgs 质量为何远小于 Planck 质量 |

### 2. 可能的新相互作用

| 新相互作用 | 理论框架 | 实验探针 |
|-----------|---------|---------|
| 超对称（SUSY） | 费米子-玻色子对称性 | LHC 超对称粒子搜索 |

| 右手中微子 | $SU(2)_R$ 或 seesaw 机制 | 无中微子双 $\beta$ 衰变 |
| 复合 Higgs | Higgs 是束缚态 | LHC 精密测量 |
