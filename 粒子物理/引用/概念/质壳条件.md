# 质壳条件（Mass Shell Condition）

---

## 一、从狭义相对论出发

### 1.1 相对论能量-动量关系

在狭义相对论中，一个质量为 $m$、动量为 $\mathbf{p}$ 的粒子，其能量为：

$$E^2 = \mathbf{p}^2 c^2 + m^2 c^4$$

在自然单位制（$c = 1$）中：

$$\boxed{E^2 - \mathbf{p}^2 = m^2}$$

### 1.2 四维形式

引入四维动量：

$$p^\mu = (E,\; \mathbf{p}) = (p^0,\; p^1,\; p^2,\; p^3)$$

Minkowski 度规 $\eta_{\mu\nu} = \text{diag}(+1, -1, -1, -1)$，则：

$$\boxed{p^\mu p_\mu = m^2}$$

展开验证：

$$p^\mu p_\mu = \eta_{\mu\nu} p^\mu p^\nu = E^2 - \mathbf{p}^2 = m^2 \quad \checkmark$$

> 这就是**质壳条件**（mass shell condition）。

---

## 二、什么是"质壳"？

### 2.1 几何图像

在四维动量空间 $(p^0, p^1, p^2, p^3)$ 中，方程

$$E^2 - p_x^2 - p_y^2 - p_z^2 = m^2$$

描述了一张**三维双曲面**（mass hyperboloid）：

```
        E (p⁰)
        ↑
        |      ╱  E = +√(p² + m²)  （粒子分支）
        |    ╱
        |  ● ← 静止粒子：p = 0, E = m
        |
  ──────┼──────────→ |p|
        |
        |  ● ← 静止反粒子：p = 0, E = -m
        |    ╲
        |      ╲  E = -√(p² + m²)  （反粒子分支）
```

这张曲面在物理中被称为**质壳**（mass shell）。

### 2.2 "壳"的含义

| 概念 | 含义 |
|:---|:---|
| **在壳（on-shell）** | 满足 $p^2 = m^2$，即物理粒子 |
| **离壳（off-shell）** | 不满足 $p^2 = m^2$，即虚粒子 |

> **质壳就是四维动量空间中"物理粒子"所占据的那张曲面。**

---

## 三、在壳与离壳

### 3.1 在壳粒子（On-shell）

真实的、可观测的粒子必须满足质壳条件：

$$p^\mu p_\mu = m^2$$

这意味着粒子的四维动量**被约束在质壳上**，四个分量 $(E, p_x, p_y, p_z)$ 中只有**三个是独立的**——能量由动量决定：

$$E = \sqrt{\mathbf{p}^2 + m^2} \quad (\text{正能解，粒子})$$

$$E = -\sqrt{\mathbf{p}^2 + m^2} \quad (\text{负能解，反粒子})$$

### 3.2 离壳粒子（Off-shell）

在量子场论的 Feynman 图中，**内线**（internal lines）代表**虚粒子**，它们不满足质壳条件：

$$p^2 \neq m^2$$

例如，在电子-电子散射（Møller 散射）中：

```
    e⁻ ──────→ e⁻
         │
         │  虚光子（离壳！q² ≠ 0）
         │
    e⁻ ──────→ e⁻
```

虚光子携带四动量 $q^\mu$，满足 $q^2 \neq 0$（即使光子质量为零）。

### 3.3 对比

| | 在壳粒子 | 离壳（虚）粒子 |
|:---:|:---:|:---:|
| 质壳条件 | $p^2 = m^2$ | $p^2 \neq m^2$ |
| 可否直接观测 | **可以** | **不可以** |
| 存在于 | Feynman 图外线 | Feynman 图内线 |
| 寿命 | 无限（稳定）或有限但真实 | $\sim \hbar / \Delta E$（不确定原理） |
| 传播子 | $\delta(p^2 - m^2)$ | $\dfrac{i}{p^2 - m^2 + i\varepsilon}$ |

## 四、质壳条件的各种等价形式

### 4.1 四维形式

$$p^\mu p_\mu = m^2$$

### 4.2 分量形式

$$E^2 - p_x^2 - p_y^2 - p_z^2 = m^2$$

### 4.3 协变 Klein-Gordon 方程

对波函数 $\phi(x)$，质壳条件对应：

$$(\partial^\mu \partial_\mu + m^2)\phi = 0$$

$$\boxed{(\Box + m^2)\phi = 0}$$

其中 $\Box = \partial^\mu\partial_\mu = \partial_t^2 - \nabla^2$ 是 d'Alembert 算子。

**验证**：平面波解 $\phi = e^{-ip\cdot x}$ 代入：

$$(\Box + m^2)e^{-ip\cdot x} = (-p^2 + m^2)e^{-ip\cdot x} = 0$$

$$\Longrightarrow \quad p^2 = m^2 \quad \checkmark$$

> **Klein-Gordon 方程就是质壳条件在坐标空间中的表述。**

### 4.4 Dirac 方程中的体现

对自旋 $1/2$ 粒子，Dirac 方程为：

$$(i\gamma^\mu \partial_\mu - m)\psi = 0$$

对平面波 $\psi = u(p)\, e^{-ip\cdot x}$：

$$(\gamma^\mu p_\mu - m)\, u(p) = 0$$

左乘 $(\gamma^\nu p_\nu + m)$：

$$(\gamma^\nu p_\nu + m)(\gamma^\mu p_\mu - m)\, u = (p^2 - m^2)\, u = 0$$

其中用到了 $\{\gamma^\mu, \gamma^\nu\} = 2\eta^{\mu\nu}$。

$$\boxed{p^2 = m^2}$$

> **Dirac 方程蕴含了质壳条件。** 实际上，Dirac 方程是 Klein-Gordon 方程的"平方根"。

### 4.5 能量-动量色散关系

$$\omega^2 = \mathbf{k}^2 + m^2$$

（自然单位制，$\omega = E$，$\mathbf{k} = \mathbf{p}$）

这就是粒子的**色散关系**——频率与波矢的关系。

## 五、特殊情况

### 5.1 无质量粒子（$m = 0$）

$$p^\mu p_\mu = 0 \quad \Longrightarrow \quad E = |\mathbf{p}|$$

> 光子、胶子等无质量粒子的四动量是**零矢量**（lightlike/null vector）。

轨迹在时空中是**零测地线**（光锥上的世界线）。

### 5.2 静止粒子（$\mathbf{p} = 0$）

$$E^2 = m^2 \quad \Longrightarrow \quad E = m$$

> 这就是 Einstein 质能关系 $E = mc^2$ 的来源——它是质壳条件在 $\mathbf{p} = 0$ 时的特殊情况。

### 5.3 超光速/快子（$m^2 < 0$，纯理论假设）

$$p^\mu p_\mu = -|m|^2 \quad \Longrightarrow \quad E^2 = \mathbf{p}^2 - |m|^2$$

- 要求 $|\mathbf{p}| > |m|$，即**永远超光速**
- 存在严重的因果性和稳定性问题
- **目前没有实验证据支持快子存在**

## 六、在量子场论中的核心作用

### 6.1 传播子（Propagator）

自由标量场的 Feynman 传播子：

$$\tilde{D}_F(p) = \frac{i}{p^2 - m^2 + i\varepsilon}$$

- **极点**在 $p^2 = m^2$，即质壳上
- 物理粒子从极点"涌现"
- $i\varepsilon$（Feynman 预设）规定了传播子的边界条件（因果性）

### 6.2 LSZ 约化公式

Lehmann-Symanzik-Zimmermann（LSZ）约化公式将 S 矩阵元与 Green 函数联系起来：

$$\langle p_1 \cdots p_n \text{ out} | q_1 \cdots q_m \text{ in} \rangle = \left[\prod_i (p_i^2 - m^2)\right]\left[\prod_j (q_j^2 - m^2)\right] G^{(n+m)}(p_1,\ldots,q_m) \Big|_{\text{on-shell}}$$

> 关键操作：将外线动量**放在质壳上**（$p^2 \to m^2$），Green 函数的极点被传播子的 $(p^2 - m^2)$ 因子抵消，得到有限的散射振幅。

### 6.3 Ward 恒等式与规范理论

在 QED 中，规范不变性要求：

$$q_\mu \mathcal{M}^\mu = 0$$

其中 $\mathcal{M}^\mu$ 是含光子外线的振幅。这保证了：
- 无质量光子只有**两个物理极化**（横向），而不是四个
- 纵向和标量极化的贡献在质壳上精确抵消

### 6.4 粒子物理中的运动学约束

在 $2 \to 2$ 散射过程中，所有**外线**必须在壳：

$$p_1^2 = m_1^2, \quad p_2^2 = m_2^2, \quad p_3^2 = m_3^2, \quad p_4^2 = m_4^2$$

但**内线**（虚粒子）可以离壳：

$$q^2 = (p_1 + p_2)^2 \neq m_{\text{中间粒子}}^2$$

> 例如：$e^+ e^- \to \mu^+ \mu^-$ 过程中，虚光子的 $q^2 = s = (p_1 + p_2)^2$ 可以取任意正值，不一定等于零。

## 七、质壳条件与相对论不变性

### 7.1 Lorentz 不变性

$p^\mu p_\mu$ 是 Lorentz 标量，因此质壳条件在**所有惯性参考系**中同时成立：

$$p'^\mu p'_\mu = \Lambda^\mu{}_\alpha\, p^\alpha \cdot \Lambda_\mu{}^\beta\, p_\beta = p^\alpha p_\alpha = m^2$$

> 粒子要么在所有参考系中都在壳，要么在所有参考系中都离壳——不存在"在某个参考系中在壳而在另一个参考系中离壳"的情况。

### 7.2 质壳条件与色散关系

| 粒子类型 | 色散关系 | 质壳条件 |
|:---:|:---:|:---:|
| 自由粒子 | $E^2 = \mathbf{p}^2 + m^2$ | $p^2 = m^2$ |
| 晶格中的电子 | $E = -2t\cos(ka)$ | 修正的色散关系 |
| 等离子体中的光子 | $\omega^2 = \omega_p^2 + c^2 k^2$ | 有效质量 $m_{\text{eff}} = \hbar\omega_p/c^2$ |

> 在凝聚态物理中，"离壳"可以被理解为**有效质量**或**环境修正**。

## 八、总结

$$\boxed{p^\mu p_\mu = m^2}$$

| 层面            | 质壳条件的含义                                     |
| :------------ | :------------------------------------------ |
| **经典力学**      | 能量-动量关系 $E^2 = \mathbf{p}^2 + m^2$          |
| **几何**        | 四维动量空间中的一张三维双曲面                             |
| **场论**        | Klein-Gordon 算子的零模条件 $(\Box + m^2)\phi = 0$ |
| **粒子物理**      | 物理粒子必须满足的运动学约束                              |
| **Feynman 图** | 外线在壳，内线可以离壳                                 |
| **散射理论**      | LSZ 约化公式中取极点极限                              |
| **对称性**       | Lorentz 不变的标量方程                             |

> **一句话：质壳条件是相对论量子理论中最基本的运动学约束——它告诉我们在四维动量空间中，哪些状态代表真实的物理粒子，哪些只是量子涨落中的虚过程。**
