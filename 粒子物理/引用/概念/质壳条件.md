## 一、物理起源

### 1.1 从相对论说起

在狭义相对论中，一个自由粒子的**四维动量**定义为：

$$p^\mu = \left(\frac{E}{c},\;\boldsymbol{p}\right) = \left(\frac{E}{c},\;p^1,\;p^2,\;p^3\right)$$

其**洛伦兹不变模方**为：

$$p^\mu p_\mu = \eta_{\mu\nu}\,p^\mu p^\nu = -\frac{E^2}{c^2} + |\boldsymbol{p}|^2$$

（采用度规约定 $\eta_{\mu\nu} = \mathrm{diag}(-1,+1,+1,+1)$）

对于一个**质量为 $m$ 的自由粒子**，狭义相对论的能量-动量关系为：

$$\boxed{E^2 = |\boldsymbol{p}|^2 c^2 + m^2 c^4}$$

等价地：

$$\boxed{p^\mu p_\mu = -m^2 c^2}$$
推理见[[注释1.1]]

这就是**质壳条件**（on-shell condition）。满足此条件的粒子称为"在质壳上"（on-shell）。

> **"壳"的含义**：在四维动量空间 $(E, p^1, p^2, p^3)$ 中，条件 $E^2 = p^2c^2 + m^2c^4$ 定义了一个**三维超曲面**——双叶双曲面。这个曲面形如一个"壳"，物理粒子的四维动量必须"落"在这个壳上。

## 二、数学表述

### 2.1 自然单位制（$\hbar = c = 1$）

在粒子物理的自然单位制中，质壳条件简化为：

$$\boxed{p^\mu p_\mu = p^2 = -m^2}$$

或等价地：

$$E^2 = |\boldsymbol{p}|^2 + m^2$$

其中 $p^2 \equiv p^\mu p_\mu$ 是四维动量的洛伦兹不变量。

### 2.2 四维动量的分解

将四维动量在质壳条件下分解：

$$p^\mu = (E,\;\boldsymbol{p}), \qquad E = \sqrt{|\boldsymbol{p}|^2 + m^2}$$

- **对于有质量粒子**（$m > 0$）：$E \geq m > 0$，质壳为双曲面的**上叶**（正能量）和**下叶**（负能量）
- **对于无质量粒子**（$m = 0$）：$E = |\boldsymbol{p}|$，质壳退化为**光锥**
- **对于虚粒子**（$p^2 \neq -m^2$）：不满足质壳条件，称为"离壳"（off-shell）

### 2.3 不同度规约定

| 度规约定 | 质壳条件 | 常见使用场景 |
|---------|---------|-------------|
| $(-,+,+,+)$ | $p^\mu p_\mu = -m^2$ | 广义相对论、部分场论教材 |
| $(+,-,-,-)$ | $p^\mu p_\mu = +m^2$ | Weinberg, Peskin & Schroeder 等 |

> 注意：本文统一使用 $(-,+,+,+)$ 约定。

## 三、在量子场论中的核心角色

### 3.1 Klein-Gordon 方程——质壳条件的场论实现

对自由标量场 $\phi(x)$，其运动方程为 [[Klein-Gordon 方程]]：

$$(\Box + m^2)\,\phi(x) = 0$$

将场做平面波展开：

$$\phi(x) = \int \frac{d^3p}{(2\pi)^3}\,\frac{1}{\sqrt{2E_{\boldsymbol{p}}}}\left(a_{\boldsymbol{p}}\,e^{-ip\cdot x} + a_{\boldsymbol{p}}^\dagger\,e^{+ip\cdot x}\right)$$

代入 [[Klein-Gordon 方程]]后，要求每个平面波分量满足：

$$(-p^2 + m^2)\,e^{-ip\cdot x} = 0 \quad \Longrightarrow \quad p^2 = -m^2$$

> **物理含义**：[[Klein-Gordon 方程]]强制自由粒子的每一分量都在质壳上。这是经典场论中质壳条件的直接体现。

### 3.2 狄拉克方程

对自由旋量场 $\psi(x)$，狄拉克方程为：

$$(i\gamma^\mu\partial_\mu - m)\,\psi(x) = 0$$

对平面波解 $\psi \sim u(p)\,e^{-ip\cdot x}$：

$$(\gamma^\mu p_\mu - m)\,u(p) = 0$$

左乘 $(\gamma^\nu p_\nu + m)$：

$$(\gamma^\nu p_\nu + m)(\gamma^\mu p_\mu - m)\,u(p) = (p^2 + m^2)\,u(p) = 0$$

因此 $p^2 = -m^2$——再次得到质壳条件。

### 3.3 Maxwell 方程

对电磁场 $A_\mu$，在洛伦兹规范 $\partial_\mu A^\mu = 0$ 下：

$$\Box\,A_\mu = 0 \quad \Longrightarrow \quad p^2 = 0$$

即光子满足 $m = 0$ 的质壳条件。

## 四、费曼图中的"壳上"与"离壳"

这是质壳条件在量子场论计算中**最具操作意义**的层面。

### 4.1 外线粒子——必须在壳

在费曼图中，**外线**（external lines）代表真实的初态或末态粒子。它们是可被实验观测的渐近态，因此**必须满足质壳条件**：

$$\boxed{\text{外线粒子：}\quad p^2 = -m^2 \quad \text{（严格在壳）}}$$

这对应于：
- 入射粒子：$p^0 = +\sqrt{|\boldsymbol{p}|^2 + m^2}$
- 出射粒子：$p^0 = +\sqrt{|\boldsymbol{p}|^2 + m^2}$

### 4.2 内线（传播子）——必须离壳

在费曼图中，**内线**（internal lines）代表传播子（propagator），其四维动量由**顶点处的四维动量守恒**决定，**不受质壳条件约束**：

$$\boxed{\text{内线粒子：}\quad p^2 \neq -m^2 \quad \text{（一般离壳）}}$$

> **通俗理解**：内线代表的粒子是"虚粒子"——它们不是可被直接观测的物理粒子，而是量子场论微扰展开中的**数学中间态**。它们"借用"了能量-动量关系的偏差，但这种偏差在量子力学的不确定性原理允许的范围内。

### 4.3 传播子的结构

**标量场传播子**：

$$\widetilde{G}(p) = \frac{-i}{p^2 + m^2 - i\epsilon}$$

分母在 $p^2 = -m^2$ 处有**极点**——这正是质壳条件。

- 当 $p^2 \to -m^2$（接近质壳）时，传播子**发散**，贡献最大
- 当 $p^2 \gg m^2$（远离质壳）时，传播子被压低

> **物理图像**：虚粒子越接近质壳（越"像"真实粒子），其传播概率越大；越远离质壳，其贡献越被量子效应压低。

**费米子传播子**：

$$S_F(p) = \frac{-i(\gamma^\mu p_\mu + m)}{p^2 + m^2 - i\epsilon} = \frac{-i(\not{p} + m)}{p^2 + m^2 - i\epsilon}$$

**光子传播子**（费曼规范）：

$$D_F^{\mu\nu}(p) = \frac{-i\eta^{\mu\nu}}{p^2 - i\epsilon}$$

### 4.4 图示对比

```
外线（在壳）：                    内线（离壳）：
                                   
  ──────── p                      ──── p ────●
  实粒子                          虚粒子（传播子）
  p² = -m²                        p² ≠ -m²
  可被观测                         不可直接观测
```

## 五、光学定理——壳上条件的深刻应用

### 5.1 光学定理的表述

光学定理是 S 矩阵**幺正性**的直接结果：

$$\boxed{\mathcal{M}(a \to a) = 2E_{\text{cm}}\,p_{\text{cm}}\,\sigma_{\text{tot}} \quad \text{（向前散射振幅的虚部与总截面的关系）}}$$

更精确地：

$$\mathrm{Im}\,\mathcal{M}(a \to a) = 2E_{\text{cm}}\,|\boldsymbol{p}_{\text{cm}}|\sum_X \int d\Pi_X\,|\mathcal{M}(a \to X)|^2$$

右边求和遍历所有可能的末态 $X$，且**所有末态粒子都在壳**。

### 5.2 质壳条件的关键作用

光学定理的推导中，关键步骤涉及**中间态的完备性插入**：

$$\mathbf{1} = \sum_X \int d\Pi_X\,|X\rangle\langle X|$$

其中相空间积分：

$$d\Pi_X = \prod_{i \in X} \frac{d^3p_i}{(2\pi)^3\,2E_i}\,(2\pi)^4\,\delta^{(4)}\!\left(P_{\text{in}} - \sum_i p_i\right)$$

每个末态粒子的 $E_i = \sqrt{|\boldsymbol{p}_i|^2 + m_i^2}$——**严格在壳**。

> **深刻意义**：光学定理将散射振幅的**虚部**（量子干涉效应）与**实过程的截面**（可观测量）联系起来，而这一联系**完全依赖于质壳条件**。

## 六、LSZ 约化公式——从关联函数到 S 矩阵

### 6.1 核心思想

量子场论的基本计算对象是**关联函数**（关联泛函，Green 函数）：

$$G^{(n)}(x_1,\ldots,x_n) = \langle 0|\mathcal{T}\{\phi(x_1)\cdots\phi(x_n)\}|0\rangle$$

但实验测量的是**散射截面和衰变率**——它们由 S 矩阵元 $\langle f|S|i\rangle$ 决定。

**LSZ 约化公式**（Lehmann-Symanzik-Zimmermann, 1955）架起了两者之间的桥梁：

$$\boxed{
\langle p_1\cdots p_n\,\text{out}|q_1\cdots q_m\,\text{in}\rangle = \left(\frac{i}{\sqrt{Z}}\right)^{m+n}\prod_{k=1}^{m}\left(p_k^2+m^2\right)\prod_{l=1}^{n}\left(q_l^2+m^2\right)\,\widetilde{G}^{(m+n)}(p_1,\ldots,q_m)\Bigg|_{\substack{p_k^2\to -m^2 \\ q_l^2\to -m^2}}
}$$

### 6.2 质壳条件的关键角色

LSZ 公式的核心操作是：

1. **先计算**动量空间关联函数 $\widetilde{G}^{(n)}(p_1,\ldots,p_n)$，此时所有动量**一般离壳**
2. **再取极限** $p_k^2 \to -m^2$，将外线动量**推到质壳上**
3. 传播子极点 $(p_k^2 + m^2)$ 与关联函数中的相应极点抵消，留下的**有限值**就是 S 矩阵元

> **通俗理解**：关联函数描述的是"一切皆有可能"的量子涨落（包括离壳的虚过程）。LSZ 公式通过"剥去"外线传播子并取质壳极限，从量子涨落中**提取出可观测的散射振幅**。

## 七、质壳条件与运动学

### 7.1 两体散射的 Mandelstam 变量

对于 $2 \to 2$ 散射过程 $a + b \to c + d$，定义 [[Mandelstam 变量]]：

$$s = -(p_a + p_b)^2, \quad t = -(p_a - p_c)^2, \quad u = -(p_a - p_d)^2$$

满足约束：

$$s + t + u = \sum_i m_i^2$$

当所有外线粒子在壳时，$s$, $t$, $u$ 中只有**两个是独立的**——这是质壳条件对运动学的约束。

### 7.2 物理区域

质壳条件进一步约束 Mandelstam 变量的取值范围：

- **$s$ 道**（$s$-channel）：$s \geq (m_a + m_b)^2$，$t$ 和 $u$ 有界
- **$t$ 道**（$t$-channel）：$t \geq 0$（弹性散射），$s$ 和 $u$ 有界
- **$u$ 道**（$u$-channel）：类似分析

这些约束决定了哪些散射过程在给定能量下是**运动学允许**的。

### 7.3 阈值条件

质壳条件决定了粒子产生的**阈值能量**。例如，要产生质量为 $M$ 的粒子：

$$\sqrt{s} = E_{\text{cm}} \geq M$$

对于 $e^+e^- \to \mu^+\mu^-$：

$$\sqrt{s} \geq 2m_\mu \approx 211\;\text{MeV}$$

## 八、虚粒子与不确定性原理

### 8.1 离壳的物理意义

虚粒子不满足质壳条件 $E^2 = p^2 + m^2$，这意味着它"违反"了经典能量-动量关系。这是否与物理矛盾？

**不矛盾**——量子力学的不确定性原理允许这种偏离：

$$\Delta E\,\Delta t \gtrsim \frac{\hbar}{2}$$

一个虚粒子可以偏离质壳 $\Delta E$，只要它存在的时间不超过：

$$\Delta t \sim \frac{\hbar}{\Delta E}$$

偏离越大，存在时间越短——这正是传播子中 $(p^2 + m^2)^{-1}$ 在离壳时被压低的物理原因。

### 8.2 经典例子

**库仑势的量子场论起源**：

两个电子通过交换一个**虚光子**相互作用。虚光子的四维动量 $q^\mu$ 满足：

$$q^2 = q_0^2 - |\boldsymbol{q}|^2 \neq 0$$

在非相对论极限下，$q_0 \approx 0$，$|\boldsymbol{q}| = |\boldsymbol{q}_{\text{transfer}}|$：

$$\text{传播子} \propto \frac{1}{q^2} = \frac{1}{-|\boldsymbol{q}|^2}$$

傅里叶变换回坐标空间：

$$V(r) = \int \frac{d^3q}{(2\pi)^3}\,\frac{e^{i\boldsymbol{q}\cdot\boldsymbol{r}}}{|\boldsymbol{q}|^2} \propto \frac{1}{r}$$

这正是**库仑势**！虚光子的离壳传播产生了长程的 $1/r$ 势。

类似地，交换虚标量粒子（质量 $m$）给出**Yukawa 势**：

$$V(r) \propto \frac{e^{-mr}}{r}$$

## 九、质壳条件的量子修正

### 9.1 自能与质量重整化

在量子水平上，粒子会通过发射和重吸收虚粒子来**修正自身的传播子**。这表现为**自能**（self-energy）$\Sigma(p)$：

$$\widetilde{G}(p) = \frac{-i}{p^2 + m_0^2 + \Sigma(p) - i\epsilon}$$

其中 $m_0$ 是**裸质量**（理论参数）。

**物理质量** $m_{\text{phys}}$ 定义为传播子极点的位置：

$$\boxed{p^2 = -m_{\text{phys}}^2 \quad \Longleftrightarrow \quad m_{\text{phys}}^2 = m_0^2 + \Sigma(-m_{\text{phys}}^2)}$$

> **物理含义**：物理粒子的质量不是输入参数 $m_0$，而是包含量子修正后的传播子极点位置。质壳条件在量子水平上被**重新定义**为**重整化传播子的极点**。

### 9.2 谱表示与 Källén-Lehmann 表示

完整的传播器可以写为**谱表示**：

$$\widetilde{G}(p) = \int_0^\infty d\mu^2\,\frac{\rho(\mu^2)}{p^2 + \mu^2 - i\epsilon}$$

其中 $\rho(\mu^2)$ 是**谱函数**（spectral function），满足：

$$\rho(\mu^2) = Z\,\delta(\mu^2 - m_{\text{phys}}^2) + \sigma(\mu^2)\,\theta(\mu^2 - s_{\text{th}})$$

- 第一项：单粒子极点，权重 $Z$（波函数重整化常数，$0 < Z \leq 1$）
- 第二项：连续谱（多粒子态），从阈值 $s_{\text{th}}$ 开始

> **质壳粒子的标志**：谱函数中的 $\delta$ 函数峰——只有当 $Z > 0$ 时，该粒子才是**渐近态**中的稳定粒子。

### 9.3 不稳定粒子——复数极点

对于不稳定粒子（如 $Z$ 玻色子、$\Delta$ 重子等），传播子的极点移到**复 $s$ 平面**上：

$$s_{\text{pole}} = M^2 - iM\Gamma$$

其中 $M$ 是粒子质量，$\Gamma$ 是衰变宽度。

$$\widetilde{G}(s) \propto \frac{1}{s - M^2 + iM\Gamma}$$

此时**严格意义上不存在质壳**——不稳定粒子没有精确的质量值，其质量分布具有 Breit-Wigner 形式：

$$|\widetilde{G}(s)|^2 \propto \frac{1}{(s - M^2)^2 + M^2\Gamma^2}$$

> **实验意义**：粒子物理实验中测量到的共振峰（如 LEP 上的 $Z$ 玻色子峰）正是这种 [[Breit-Wigner 分布]]。峰位给出 $M$，半高宽给出 $\Gamma$。

## 十、总结：质壳条件的多重角色

| 角色 | 具体表现 |
|------|---------|
| **运动学约束** | $p^2 = -m^2$ 确定自由粒子的能量-动量关系 |
| **场方程的推论** | Klein-Gordon / Dirac / Maxwell 方程强制在壳 |
| **外线条件** | 费曼图外线必须在壳——初末态是真实粒子 |
| **传播子极点** | 质壳对应传播子的极点，决定物理质量 |
| **LSZ 约化** | 从离壳关联函数提取在壳 S 矩阵元的桥梁 |
| **光学定理** | 幺正性要求中间态完备性求和中的粒子在壳 |
| **阈值条件** | 决定哪些散射/产生过程在运动学上被允许 |
| **量子修正** | 重整化后的质壳极点给出物理质量 |
| **不稳定粒子** | 极点移到复平面，质壳概念推广为共振 |

用一句话概括：

$$\boxed{\text{质壳条件} = \text{相对论性自由粒子的运动学约束} = \text{传播子的极点} = \text{真实粒子的标志}}$$
