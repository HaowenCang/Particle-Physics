## 一、为什么需要快度？

### 1. 速度的问题

在相对论中，粒子速度 $\beta = v/c$ 的取值范围是 $[0, 1)$——一个**有界**的区间。这在粒子物理中带来了实际困难：

- 粒子的速度可以在 Lorentz boost 下改变，但永远不能超过 $c$。
- 两个接近光速的粒子的"相对速度"并不直观——简单的 $v_{\text{rel}} = v_1 - v_2$ 不再适用。
- 在对撞机物理中，几乎所有粒子都以接近光速运动，用 $\beta$ 来区分它们很不方便。

**快度**（rapidity）的引入解决了这些问题：它提供了一个**无界**的、在 Lorentz boost 下**平移**的运动学变量。

> 如果把速度比作"温度计"（有上下限），快度就像一个刻度无限延伸的"温度标尺"——你可以自由地加减，而不用担心撞到边界。

---

## 二、快度的定义

### 1. 一维运动学

考虑粒子沿 $z$ 方向运动。定义快度 $y$：

$$\boxed{y = \frac{1}{2}\ln\frac{E + p_z}{E - p_z} = \mathrm{artanh}\,\beta_z}$$

其中 $E = p^0$ 是能量，$p_z$ 是 $z$ 方向动量分量。

**等价形式**：

$$y = \frac{1}{2}\ln\frac{1+\beta_z}{1-\beta_z} = \mathrm{artanh}\,\beta_z$$

反解：

$$\beta_z = \tanh y, \qquad \gamma = \cosh y, \qquad \gamma\beta_z = \sinh y$$

### 2. 四动量的快度参数化

利用快度，粒子的四动量可以写为：

$$\boxed{p^\mu = (E, \, p_x, \, p_y, \, p_z) = (m_T\cosh y, \, p_x, \, p_y, \, m_T\sinh y)}$$

其中**横质量**（transverse mass）定义为：

$$m_T = \sqrt{m^2 + p_T^2}, \qquad p_T = \sqrt{p_x^2 + p_y^2}$$

验证：

$$E^2 - p_z^2 = m_T^2(\cosh^2 y - \sinh^2 y) = m_T^2 = m^2 + p_T^2$$

因此：

$$E^2 = p_z^2 + m_T^2 = m_T^2\cosh^2 y$$

> 快度将粒子的纵向运动（$z$ 方向）和横向运动（$x, y$ 方向）干净地分离——纵向由 $y$ 描述，横向由 $p_T$ 和方位角 $\phi$ 描述。这种分离在对撞机物理中极为自然，因为对撞束流沿纵向运动，而新粒子的产生主要体现在横向上。

## 三、快度的核心性质：Lorentz boost 下的平移

### 1. Lorentz boost 的快度表示

沿 $z$ 方向、速度为 $\beta_b$ 的 Lorentz boost 对应快度参数：

$$y_b = \mathrm{artanh}\,\beta_b$$

在快度语言中，**Lorentz boost 就是快度的平移**：

$$\boxed{y' = y - y_b}$$

### 2. 证明

在 boost 下，四动量变换为：

$$E' = \gamma_b(E - \beta_b p_z), \qquad p_z' = \gamma_b(p_z - \beta_b E)$$

因此：

$$y' = \frac{1}{2}\ln\frac{E' + p_z'}{E' - p_z'} = \frac{1}{2}\ln\frac{\gamma_b(1-\beta_b)(E+p_z)}{\gamma_b(1+\beta_b)(E-p_z)}$$

$$= \frac{1}{2}\ln\frac{E+p_z}{E-p_z} + \frac{1}{2}\ln\frac{1-\beta_b}{1+\beta_b} = y - y_b \quad \square$$

> 速度在 Lorentz boost 下的变换是非线性的（相对论速度叠加公式），计算很繁琐。但快度的变换就是简单的加减法——这正是引入快度的最大动机。

### 3. 快度的加法

如果粒子 1 的快度为 $y_1$，粒子 2 的快度为 $y_2$，则它们的**快度差**：

$$\Delta y = y_1 - y_2$$

是 **Lorentz 不变量**（在沿 $z$ 方向的 boost 下不变）。

两个粒子的**相对快度**：

$$y_{\text{rel}} = y_1 - y_2$$

对应相对速度：

$$\beta_{\text{rel}} = \tanh|y_1 - y_2|$$

## 四、[[赝快度]]（Pseudorapidity）

### 1. 定义

在实验中，粒子的能量和动量的纵向分量往往不能直接测量，但散射角 $\theta$（粒子动量与束流方向的夹角）可以直接测量。由此定义**赝快度**：

$$\boxed{\eta = -\ln\tan\frac{\theta}{2} = \mathrm{artanh}(\cos\theta)}$$

### 2. 与快度的关系

在极端相对论极限（$m \ll E$，即 $E \approx |\mathbf{p}|$）下：

$$y = \frac{1}{2}\ln\frac{E + p_z}{E - p_z} \approx \frac{1}{2}\ln\frac{|\mathbf{p}| + |\mathbf{p}|\cos\theta}{|\mathbf{p}| - |\mathbf{p}|\cos\theta} = \frac{1}{2}\ln\frac{1+\cos\theta}{1-\cos\theta} = -\ln\tan\frac{\theta}{2} = \eta$$

因此：

$$\boxed{y \approx \eta \quad \text{当} \quad m \ll E}$$

对于有质量粒子，两者有差异：

$$y - \eta \approx \frac{m^2}{2p_T^2}\cdot\frac{1}{\cosh^2\eta} + \cdots$$

### 3. 赝快度与散射角的对应

| $\theta$ | $\eta$ | 物理位置 |
|-----------|--------|---------|
| $0°$ | $+\infty$ | 沿束流方向（前向） |
| $45°$ | $\approx 0.88$ | — |
| $90°$ | $0$ | 垂直于束流（中心区） |
| $135°$ | $\approx -0.88$ | — |
| $180°$ | $-\infty$ | 沿束流反方向（后向） |

### 4. 用赝快度表示三维动量

$$\mathbf{p} = (p_T\cos\phi, \; p_T\sin\phi, \; p_T\sinh\eta)$$

$$|\mathbf{p}| = \frac{p_T}{\sin\theta} = p_T\cosh\eta$$

$$E \approx |\mathbf{p}| = p_T\cosh\eta \quad (\text{无质量或极端相对论})$$

> 赝快度是实验物理学家最常用的变量——它直接从探测器中的径迹角度就能算出来，不需要知道粒子的质量。$\eta$ 的均匀间隔对应探测器中**均匀的角间距**（在 $\eta$ 而非 $\theta$ 上），这使得探测器的设计和数据分析在 $\eta$ 坐标下特别方便。

## 五、横动量（Transverse Momentum）与 $(y, p_T, \phi)$ 坐标系

### 1. 完整的运动学变量

在对撞机物理中，一个末态粒子由以下变量完全描述：

| 变量 | 定义 | 取值范围 |
|------|------|---------|
| $y$（或 $\eta$） | 快度（或赝快度） | $(-\infty, +\infty)$ |
| $p_T$ | 横动量大小 | $[0, \infty)$ |
| $\phi$ | 方位角（围绕束流轴） | $[0, 2\pi)$ |
| $m$ | 粒子质量 | 固定值 |

### 2. 洛伦兹不变相空间

单粒子洛伦兹不变相空间在 $(y, p_T, \phi)$ 坐标下为：

$$\frac{d^3p}{(2\pi)^3 2E} = \frac{p_T\,dp_T\,d\phi\,dy}{2(2\pi)^3}$$

推导：$d^3p = dp_x\,dp_y\,dp_z = p_T\,dp_T\,d\phi\,dp_z$，且 $dp_z = \dfrac{E}{p_z}\,dp_z + \cdots$ 利用 $E\,dE = p_z\,dp_z$（在固定 $p_T$ 时），得 $dp_z/E = dy$，因此：

$$\frac{d^3p}{2E} = \frac{p_T\,dp_T\,d\phi\,dy}{2}$$

> 这个结果非常优美——在 $(y, p_T, \phi)$ 坐标下，相空间积分的被积函数中**没有**复杂的运动学因子，只有简单的 $p_T/2$。这使得微分散射截面在快度坐标下的计算和物理分析都非常干净。

## 六、快度在对撞机物理中的应用

### 1. 快度分布（Rapidity Distribution）

粒子产生的微分散射截面通常表示为：

$$\frac{d\sigma}{dy\,dp_T^2\,d\phi}$$

对 $p_T$ 和 $\phi$ 积分后得到**快度分布** $dN/dy$：

$$\frac{dN}{dy} = \int dp_T^2 \int d\phi\;\frac{d\sigma}{dy\,dp_T^2\,d\phi}$$

**快度分布的形状**携带了丰富的物理信息：

- **中心区**（$y \approx 0$）：由"软"QCD过程主导，粒子多重数密度近似平坦
- **前向区**（$|y|$ 大）：反映入射粒子的碎裂，与部分子分布函数相关
- **快度分布的宽度**：与碰撞能量和粒子质量有关

### 2. Drell-Yan 过程的快度分布

考虑 $q\bar{q} \to \gamma^*/Z \to l^+l^-$（Drell-Yan 过程）。产生的轻子对的不变质量为 $M$，快度为 $Y$。

在部分子模型中：

$$\frac{d\sigma}{dM^2\,dY} = \frac{4\pi\alpha^2}{3sM^2}\sum_q e_q^2\left[f_q(x_1)\,f_{\bar{q}}(x_2) + (q \leftrightarrow \bar{q})\right]$$

其中部分子动量分数：

$$\boxed{x_1 = \frac{M}{\sqrt{s}}\,e^{+Y}, \qquad x_2 = \frac{M}{\sqrt{s}}\,e^{-Y}}$$

> 轻子对的快度 $Y$ 直接反映了碰撞中正反夸克的动量分数比 $x_1/x_2 = e^{2Y}$。通过测量 $Y$ 的分布，我们可以反推夸克在质子中的动量分布（PDF），这是快度在实验中最强大的应用之一。

### 3. 快度间隙（Rapidity Gap）与衍射散射

在**衍射散射**（diffractive scattering）过程中，末态中出现**无粒子产生的快度区间**：

```
快度 y
←─────────────────────────────────→
  末态粒子  │   快度间隙   │  末态粒子
  (前向)    │  (无粒子)    │  (中心)
```

这种快度间隙是**Pomeron 交换**的实验标志——衍射过程中，质子保持完整（或仅被激发为低质量态），中间留下一个没有粒子产生的区域。

> 在"普通"碰撞中，粒子在所有快度区间都有产生。但衍射碰撞中，两束粒子"穿过"彼此而不完全碎裂，留下一个"空白"区域。快度间隙的存在直接反映了 QCD 中色单态交换的机制。

## 七、光锥坐标与快度

### 1. 光锥分量

定义光锥坐标：

$$p^+ = \frac{E + p_z}{\sqrt{2}}, \qquad p^- = \frac{E - p_z}{\sqrt{2}}$$

快度与光锥分量的关系：

$$y = \frac{1}{2}\ln\frac{p^+}{p^-}$$

因此：

$$p^+ = m_T\,e^{+y}/\sqrt{2}, \qquad p^- = m_T\,e^{-y}/\sqrt{2}$$

### 2. Lorentz boost 的作用

沿 $z$ 方向的 boost（快度 $y_b$）：

$$p^{+\prime} = e^{+y_b}\,p^+, \qquad p^{-\prime} = e^{-y_b}\,p^-$$

即 $p^+$ 和 $p^-$ 分别被 $e^{\pm y_b}$ 缩放。这正是快度平移 $y' = y - y_b$ 的光锥表述。

### 3. 不变量用光锥分量表示

$$p^\mu p_\mu = 2p^+p^- - p_T^2 = m^2$$

$$s = (p_1 + p_2)^2 = 2p_1^+p_2^- + 2p_1^-p_2^+ - 2\mathbf{p}_{1T}\cdot\mathbf{p}_{2T}$$

在无质量极限或对头碰中（$p_{1T} = p_{2T} = 0$）：

$$s = 2p_1^+p_2^- + 2p_1^-p_2^+$$

## 八、质心系快度与阈值行为

### 1. 两粒子质心系

对于对头碰的两粒子（$m_1 = m_2 = m$，能量 $E$），质心系中：

$$y_1^{\text{cm}} = +y_0, \qquad y_2^{\text{cm}} = -y_0$$

其中：

$$y_0 = \mathrm{artanh}\frac{p^*}{E^*} = \mathrm{artanh}\sqrt{1 - \frac{4m^2}{s}}$$

在极端相对论极限（$\sqrt{s} \gg m$）：

$$y_0 \approx \ln\frac{\sqrt{s}}{m}$$

### 2. 实验室系快度

在实验室系（靶粒子静止）中，入射粒子的快度：

$$y_{\text{lab}} = \mathrm{artanh}\frac{p_{\text{lab}}}{E_{\text{lab}}} = \mathrm{artanh}\sqrt{1 - \frac{m^2}{E_{\text{lab}}^2}}$$

在极端相对论极限：$y_{\text{lab}} \approx \ln\dfrac{2E_{\text{lab}}}{m}$。

### 3. 快度的可用范围

在质心系中，末态粒子的快度被限制在：

$$|y| \leq y_{\max} \approx \frac{1}{2}\ln\frac{s}{m_T^2}$$

随着碰撞能量 $\sqrt{s}$ 增加，可用的快度范围以 $\ln s$ 的速度增长——这就是所谓的**快度"纵向空间"的扩展**。

## 九、快度分布的典型特征

### 1. 质子-质子碰撞中的带电粒子快度分布

实验观测到的 $dN_{\text{ch}}/d\eta$ 分布呈现特征性的**"台地"形状**：

```
dN/dη
  ↑
  │      ╭──────────────────╮
  │    ╭─╯                  ╰─╮
  │  ╭─╯                      ╰─╮
  │──╯                            ╰──
  └──────────────────────────────── η →
      -y_beam    0    +y_beam
```

- **中心区**（$\eta \approx 0$）：分布近似平坦，高度随 $\sqrt{s}$ 缓慢增长（$\sim s^{0.1}$）
- **碎裂区**（$|\eta| \sim y_{\text{beam}}$）：分布下降，反映入射粒子的碎裂
- 台地的**宽度**随 $\ln s$ 增长

### 2. 能量依赖

中心区的粒子多重数增长近似服从：

$$\left.\frac{dN_{\text{ch}}}{d\eta}\right|_{\eta=0} \propto s^{0.11} \quad \text{或} \quad \propto (\ln s)^2$$

不同模型给出不同的增长速率，这仍是 QCD 中未完全解决的问题。

## 十、快度与 Feynman-$x$

### 1. Feynman-$x_F$ 的定义

Feynman-$x$（或 $x_F$）是纵向动量在质心系中的无量纲度量：

$$\boxed{x_F = \frac{2p_z^{\text{cm}}}{\sqrt{s}}}$$

其中 $p_z^{\text{cm}}$ 是末态粒子在质心系中的纵向动量。

### 2. 与快度的关系

在极端相对论极限（$m \ll m_T \ll \sqrt{s}$）下：

$$x_F \approx \frac{2m_T\sinh y}{\sqrt{s}} \approx \frac{m_T}{\sqrt{s}}\,e^{y} \quad (y \gg 1)$$

更精确地：

$$x_F = \frac{m_T}{\sqrt{s}}\left(e^{y} - e^{-y}\right) = \frac{2m_T\sinh y}{\sqrt{s}}$$

### 3. 限制

$$|x_F| \leq x_{F,\max} = \frac{2p_z^{\text{cm,max}}}{\sqrt{s}} \approx \frac{2m_T}{\sqrt{s}}\sinh y_{\max} \leq 1$$

$x_F = +1$ 对应"最大前向"粒子（以质心系中接近光速向前运动），$x_F = -1$ 对应"最大后向"。

> $x_F$ 度量的是末态粒子携带了多少入射粒子的纵向动量。$x_F$ 接近 1 的粒子"记住"了入射粒子的方向和动量，而 $x_F \approx 0$ 的粒子是在碰撞中心区"新产生"的。

## 十一、重离子碰撞中的快度分布

### 1. Bjorken 快度密度

在相对论重离子碰撞（如 RHIC、LHC 的铅-铅碰撞）中，**Bjorken** 提出了一个简单但深远的估计——中心快度区的能量密度：

$$\boxed{\varepsilon_{\text{Bj}} = \frac{1}{A_T\,\tau_0}\frac{dE_T}{dy}\bigg|_{y=0}}$$

其中：
- $A_T$ 是碰撞重叠区的横截面积
- $\tau_0$ 是热化时间（$\sim 1$ fm/$c$）
- $dE_T/dy$ 是单位快度的横能量

在 LHC 的 Pb-Pb 碰撞中（$\sqrt{s_{NN}} = 5.02$ TeV）：

$$\varepsilon_{\text{Bj}} \sim 12\text{--}15 \;\text{GeV/fm}^3$$

远超 QCD 相变的临界能量密度（$\varepsilon_c \sim 0.5\text{--}1$ GeV/fm$^3$），确认了**夸克-胶子等离子体**（QGP）的形成。

### 2. 快度分布的集体流

QGP 中的集体膨胀在快度分布上留下印记：
- **纵向流**（Hubble-like expansion）：导致快度分布的展宽
- **横向流**：影响 $p_T$ 分布，但与快度分布间接关联

## 十二、快度的实验测量

### 1. 直接测量

- **径迹探测器**（硅像素/条带探测器）：测量带电粒子的径迹方向 $\Rightarrow$ $\eta$ 和 $\phi$
- **量能器**（电磁/强子量能器）：测量能量沉积的位置 $\Rightarrow$ $\eta$ 和 $\phi$

### 2. 快度 vs 赝快度的实验差异

对于已知质量的粒子（如 $\pi^\pm$、$K^\pm$、$p$），可以从测量的 $\mathbf{p}$ 计算真正的快度：

$$y = \frac{1}{2}\ln\frac{E+p_z}{E-p_z} = \frac{1}{2}\ln\frac{\sqrt{m^2+p_T^2\cosh^2\eta}+p_T\sinh\eta}{\sqrt{m^2+p_T^2\cosh^2\eta}-p_T\sinh\eta}$$

差异：

$$y - \eta = \frac{1}{2}\ln\frac{\sqrt{m^2+p_T^2\cosh^2\eta}+p_T\sinh\eta}{\sqrt{m^2+p_T^2}-p_T\sinh\eta \cdot \text{(简化形式)}}$$

在 $p_T \gg m$ 时差异可忽略；在 $p_T \sim m$ 时差异显著。

### 3. 典型探测器的赝快度覆盖

| 实验         | 探测器     | $\| \eta \|$ 覆盖    |
| ---------- | ------- | ------------------ |
| ATLAS（LHC） | 内部径迹探测器 | $\| \eta \| < 2.5$ |
| ATLAS      | 量能器     | $\| \eta \| < 4.9$ |
| CMS（LHC）   | 全部系统    | $\| \eta \| < 5.0$ |
| LHCb（LHC）  | 前向谱仪    | $2 < \|\eta\| < 5$     |
| ALICE（LHC） | 中心桶部    | $\| \eta \| < 0.9$ |
