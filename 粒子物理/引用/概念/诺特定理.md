## 一、历史背景与核心思想

### 1.1 历史渊源

1915年，**大卫·希尔伯特**（David Hilbert）在建立广义相对论的变分原理时，已经注意到能量-动量守恒与坐标变换不变性之间的联系。1918年，**埃米·诺特**（Emmy Noether）——当时哥廷根大学最杰出的数学家之一——将这一观察提升为一个**普适定理**，发表在论文 *"Invariante Variationsprobleme"* 中。

### 1.2 核心思想

诺特定理的本质可以用一句话概括：

$$\boxed{\text{每一个连续对称性都对应一个守恒量，反之亦然}}$$

## 二、数学表述

### 2.1 设置

考虑一个经典场论，基本变量为场 $\phi^a(x)$（$a$ 标记场的内部指标），作用量为：

$$S[\phi] = \int_\Omega \mathcal{L}\!\left(\phi^a,\;\partial_\mu\phi^a,\;x\right)\,d^4x$$

其中 $\mathcal{L}$ 是拉格朗日密度，$\Omega$ 是时空积分区域。

### 2.2 无穷小变换

考虑一个含参数 $\epsilon$ 的无穷小变换：

$$\begin{cases} x^\mu \;\to\; x'^\mu = x^\mu + \delta x^\mu \\[4pt] \phi^a(x) \;\to\; \phi'^a(x') = \phi^a(x) + \delta\phi^a(x) \end{cases}$$

其中 $\delta x^\mu$ 和 $\delta\phi^a$ 都是 $\mathcal{O}(\epsilon)$ 的无穷小量。

为方便，定义**场在同一点的变化**（Lie 变分）：

$$\bar{\delta}\phi^a(x) \equiv \phi'^a(x) - \phi^a(x) = \delta\phi^a - (\partial_\mu\phi^a)\,\delta x^\mu$$

### 2.3 作用量的变分

在变换下，作用量的变化为：

$$\delta S = \int_{\Omega'} \mathcal{L}'(\phi',\partial'\phi',x')\,d^4x' - \int_\Omega \mathcal{L}(\phi,\partial\phi,x)\,d^4x$$

经过标准的变分计算（利用雅可比行列式和分部积分），可以证明：

$$\delta S = \int_\Omega \left[\frac{\delta S}{\delta\phi^a}\,\bar{\delta}\phi^a + \partial_\mu\!\left(\mathcal{L}\,\delta x^\mu + \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^a)}\,\bar{\delta}\phi^a\right)\right]d^4x$$

其中第一项是**运动方程**的贡献：

$$\frac{\delta S}{\delta\phi^a} = \frac{\partial\mathcal{L}}{\partial\phi^a} - \partial_\mu\frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^a)}$$

在壳层（on-shell）上，即满足运动方程时，这一项为零。

### 2.4 诺特定理的精确表述

**定理**：若作用量在无穷小变换 $(\delta x^\mu, \delta\phi^a)$ 下**不变**（$\delta S = 0$），则在满足运动方程的解上，存在守恒流：

$$\boxed{J^\mu = \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^a)}\,\bar{\delta}\phi^a + \mathcal{L}\,\delta x^\mu}$$

满足**连续性方程**：

$$\partial_\mu J^\mu = 0$$

对应的**守恒荷**为：

$$Q = \int_\Sigma J^0\,d^3x$$

满足：

$$\frac{dQ}{dt} = 0$$

### 2.5 实用形式——参数化的对称变换

实际应用中，对称变换通常写为含参数 $\epsilon^A$ 的形式（$A$ 标记独立的对称变换）：

$$\begin{cases} \delta x^\mu = \epsilon^A\,X^\mu_A(x) \\[4pt] \delta\phi^a = \epsilon^A\,\Psi^a_A(\phi,x) \end{cases}$$

此时对应的诺特流为：

$$\boxed{J^\mu_A = \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^a)}\,\Psi^a_A + \mathcal{L}\,X^\mu_A}$$

每个独立参数 $\epsilon^A$ 对应一个独立的守恒流 $J^\mu_A$。

## 三、核心应用

### 3.1 空间平移对称性 → 动量守恒

**对称变换**：$\phi^a(x) \to \phi^a(x + \boldsymbol{\epsilon})$，即

$$\delta x^\mu = \epsilon^\mu \quad (\text{常数}), \qquad \bar{\delta}\phi^a = 0$$

**诺特流**：

$$J^\mu_\nu = \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^a)}\,\partial_\nu\phi^a - \mathcal{L}\,\delta^\mu_\nu$$

这正是**能量-动量张量**（Canonical Energy-Momentum Tensor）：

$$\boxed{T^\mu_{\ \nu} = \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^a)}\,\partial_\nu\phi^a - \mathcal{L}\,\delta^\mu_\nu}$$

守恒律 $\partial_\mu T^\mu_{\ \nu} = 0$ 给出：

- $\nu = 0$：$\partial_\mu T^\mu_{\ 0} = 0$ → **能量守恒**，$E = \int T^{00}\,d^3x$
- $\nu = i$：$\partial_\mu T^\mu_{\ i} = 0$ → **动量守恒**，$P_i = \int T^{0}_{\ i}\,d^3x$

### 3.2 时间平移对称性 → 能量守恒

从 3.1 中 $\nu = 0$ 分量单独提取：

$$\partial_\mu T^{\mu 0} = 0 \quad \Longrightarrow \quad \frac{d}{dt}\int T^{00}\,d^3x = 0$$

$T^{00}$ 就是能量密度，其空间积分是总能量——守恒。

### 3.3 空间旋转对称性 → 角动量守恒

**对称变换**：无穷小旋转 $\delta x^i = \epsilon^{ij}x^j$（$\epsilon^{ij} = -\epsilon^{ji}$），场做相应的洛伦兹旋转：

$$\bar{\delta}\phi^a = -\frac{i}{2}\,\epsilon^{ij}\,(S^{ij})^a_{\ b}\,\phi^b$$

其中 $S^{ij}$ 是场的自旋矩阵。

**诺特流**（反对称张量）：

$$M^{\mu ij} = x^i T^{\mu j} - x^j T^{\mu i} + \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^a)}\,(S^{ij})^a_{\ b}\,\phi^b$$

**守恒荷**：

$$\boxed{J^{ij} = \int \!\left(x^i T^{0j} - x^j T^{0i}\right)d^3x + \int \pi^a\,(S^{ij})^a_{\ b}\,\phi^b\,d^3x}$$

第一项是**轨道角动量**，第二项是**自旋角动量**。

### 3.4 $\mathrm{U(1)}$ 全局对称性 → 电荷守恒

**对称变换**：$\psi \to e^{i\alpha}\psi$，即

$$\delta\psi = i\alpha\,\psi, \qquad \delta\bar{\psi} = -i\alpha\,\bar{\psi}$$

对于狄拉克拉格朗日量 $\mathcal{L} = \bar{\psi}(i\gamma^\mu\partial_\mu - m)\psi$：

**诺特流**：

$$\boxed{J^\mu = \bar{\psi}\,\gamma^\mu\,\psi}$$

**守恒荷**：

$$Q = \int \psi^\dagger\psi\,d^3x = \int \bar{\psi}\gamma^0\psi\,d^3x$$

这就是**电荷**（或粒子数）守恒的场论表述。

> **与规范耦合的联系**：在上一节中我们看到，将这个 $\mathrm{U(1)}$ 对称性从全局推广到局域，就必须引入规范场 $A_\mu$。而诺特流 $J^\mu$ 正是规范场的**源**——它出现在 Maxwell 方程的右边：$\partial_\nu F^{\mu\nu} = g\,J^\mu$。

### 3.5 规范对称性 → 高斯定律

对于**局域**规范变换 $\alpha = \alpha(x)$，诺特定理的推广给出的不是新的守恒荷，而是**约束方程**（高斯定律）：

$$\nabla \cdot \boldsymbol{E} = \rho$$

这是运动方程的一个分量，不是由诺特定理给出的守恒律，而是规范对称性施加的**约束**。

## 四、相对论场论中的协变表述

### 4.1 四维角动量的统一

在洛伦兹协变的理论中，时空平移和洛伦兹变换统一为**庞加莱变换**：

$$x'^\mu = \Lambda^\mu_{\ \nu}\,x^\nu + a^\mu$$

10 个参数（4 个平移 + 6 个洛伦兹变换）对应 10 个守恒量：

| 对称变换 | 参数个数 | 守恒量 |
|---------|---------|--------|
| 时间平移 | 1 | 能量 $E$ |
| 空间平移 | 3 | 动量 $P^i$ |
| 空间旋转 | 3 | 角动量 $J^i$ |
| 洛伦兹推进 | 3 | 质心运动定理 $K^i = tP^i - x^i E$ |

### 4.2 诺特流的张量结构

对于庞加莱群，诺特流构成一个二阶张量：

$$\mathcal{J}^{\mu(\lambda\rho)} = x^\lambda T^{\mu\rho} - x^\rho T^{\mu\lambda} + \Sigma^{\mu\lambda\rho}$$

其中 $\Sigma^{\mu\lambda\rho}$ 是自旋张量贡献。守恒律 $\partial_\mu\mathcal{J}^{\mu(\lambda\rho)} = 0$ 统一了动量守恒和角动量守恒。

## 五、经典力学中的诺特定理

在分析力学中，诺特定理有更为直观的形式。

### 5.1 拉格朗日表述

对于广义坐标 $q^i(t)$ 和拉格朗日量 $L(q^i, \dot{q}^i, t)$，若变换

$$q^i \to q^i + \epsilon\,\eta^i(q, \dot{q}, t)$$

使作用量 $S = \int L\,dt$ 不变，则守恒量为：

$$\boxed{Q = \frac{\partial L}{\partial\dot{q}^i}\,\eta^i = p_i\,\eta^i}$$

| 对称性 | 变换 $\eta^i$ | 守恒量 $Q$ |
|--------|-------------|-----------|
| 时间平移不变 | $\dot{q}^i$（$t \to t + \epsilon$） | $H = p_i\dot{q}^i - L$（哈密顿量/能量） |
| 空间平移不变 | $1$（$q \to q + \epsilon$） | $p = \partial L/\partial\dot{q}$（正则动量） |
| 旋转不变 | $\hat{n} \times \boldsymbol{r}$ | $\boldsymbol{L} = \boldsymbol{r} \times \boldsymbol{p}$（角动量） |

### 5.2 哈密顿表述

在相空间中，对称性表现为相空间函数 $Q(q,p)$ 与哈密顿量的**泊松括号**为零：

$$\{Q, H\} = 0 \quad \Longleftrightarrow \quad \frac{dQ}{dt} = 0$$

而对称变换本身由 $Q$ 生成：

$$\delta_\epsilon f = \epsilon\,\{f, Q\}$$

> **深刻联系**：守恒量既是**对称性的标志**，又是**对称变换的生成元**——这是哈密顿力学中最优美的结构之一。

## 六、诺特定理的逆定理

诺特定理也有**逆命题**：

> **若一个系统存在守恒量 $Q$（满足 $dQ/dt = 0$），则存在一个使作用量不变的连续变换，其生成元为 $Q$。**

在哈密顿表述中，这几乎是显然的：$\{Q, H\} = 0$ 意味着 $Q$ 生成一个与时间演化对易的变换。

这使得诺特定理成为**对称性与守恒律之间的完全等价**。

## 七、推广与深化

### 7.1 准对称性（Quasi-symmetry）

有时作用量在变换下**不是严格不变**，而是变化一个**全导数项**：

$$\delta S = \epsilon\int_\Omega \partial_\mu K^\mu\,d^4x$$

此时诺特流需修正为：

$$J^\mu = \frac{\partial\mathcal{L}}{\partial(\partial_\mu\phi^a)}\,\bar{\delta}\phi^a + \mathcal{L}\,\delta x^\mu - K^\mu$$

仍然满足 $\partial_\mu J^\mu = 0$。

**实例**：电磁场的规范变换 $A_\mu \to A_\mu + \partial_\mu\Lambda$ 就是这种准对称性。

### 7.2 离散对称性

诺特定理**仅适用于连续对称性**。离散对称性（如宇称 $P$、时间反演 $T$、电荷共轭 $C$）不产生守恒流，但可以产生**选择定则**和**量子数约束**。

### 7.3 量子力学中的诺特定理

在量子力学中，守恒量 $Q$ 对应一个厄米算符，诺特定理表现为：

$$[Q, H] = 0 \quad \Longleftrightarrow \quad \frac{d\langle Q\rangle}{dt} = 0$$

对称变换由幺正算符 $U = e^{i\epsilon Q}$ 实现。

### 7.4 量子场论中的反常（Anomaly）

在量子水平上，经典对称性可能被**量子效应破坏**——这称为**反常**（anomaly）。

以 $\mathrm{U(1)_A}$ 轴矢量对称性为例：

$$\partial_\mu J_5^\mu = -\frac{e^2}{16\pi^2}\,\epsilon^{\mu\nu\rho\sigma}F_{\mu\nu}F_{\rho\sigma}$$

经典守恒律被量子修正打破。**Adler-Bell-Jackiw 反常**是理解 $\pi^0 \to \gamma\gamma$ 衰变的关键。

> **物理意义**：反常不是理论的缺陷，而是**量子理论必须满足自洽性条件**的体现。如果一个规范对称性出现反常，理论就不可重整，是不自洽的——这限制了可能的量子场论的构建。

## 八、全局图景：对称性在现代物理中的地位

诺特定理揭示的对称性-守恒律对应关系，是整个现代物理学的**组织原则**：

$$\boxed{
\begin{array}{ccc}
\text{对称性} & \xrightarrow{\text{诺特定理}} & \text{守恒律} \\[8pt]
\text{全局对称} & \longrightarrow & \text{守恒荷（能量、动量、电荷…）} \\[6pt]
\text{局域对称} & \longrightarrow & \text{规范场 + 相互作用} \\[6pt]
\text{对称性自发破缺} & \longrightarrow & \text{Goldstone 玻色子 / Higgs 机制}
\end{array}
}$$

标准模型的全部内容可以概括为：

$$\mathcal{L}_{\text{SM}} = \underbrace{-\frac{1}{4}F_{\mu\nu}F^{\mu\nu}}_{\text{规范对称性}} + \underbrace{\bar{\psi}\,i\gamma^\mu D_\mu\,\psi}_{\text{局域规范不变性}} + \underbrace{|D_\mu\Phi|^2 - V(\Phi)}_{\text{Higgs 机制}} + \underbrace{y\,\bar{\psi}\psi\,\Phi}_{\text{Yukawa 耦合}}$$

每一项的**存在**和**形式**都由对称性**唯一确定**——这正是诺特精神的体现。
