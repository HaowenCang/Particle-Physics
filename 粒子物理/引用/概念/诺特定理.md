> **每一个连续对称性，都对应一个守恒量。**

这是 1918 年由 Emmy Noether 证明的定理，被认为是理论物理中最深刻、最优美的定理之一。

---

## 数学表述

### 变分原理的框架

考虑一个作用量：

$$S = \int \mathcal{L}(\phi,\, \partial_\mu \phi,\, x)\, d^4x$$

如果对场和坐标做一个无穷小变换：

$$x^\mu \to x'^\mu = x^\mu + \delta x^\mu$$
$$\phi(x) \to \phi'(x') = \phi(x) + \delta \phi(x)$$

使得作用量**不变**（$\delta S = 0$），则存在一个守恒流：

$$\partial_\mu j^\mu = 0$$

对应的守恒荷为：

$$Q = \int j^0\, d^3x, \qquad \frac{dQ}{dt} = 0$$

---

## 守恒流的显式构造

对于拉氏量在变换下不变的情况，诺特流为：

$$\boxed{j^\mu = \frac{\partial \mathcal{L}}{\partial(\partial_\mu \phi)}\,\delta\phi - T^{\mu\nu}\,\delta x_\nu}$$

其中 $T^{\mu\nu}$ 是能量-动量张量：

$$T^{\mu\nu} = \frac{\partial \mathcal{L}}{\partial(\partial_\mu \phi)}\,\partial^\nu \phi - g^{\mu\nu}\mathcal{L}$$

## 经典例子

### 1. 时间平移对称性 → 能量守恒

$$t \to t + \delta t \quad (\mathcal{L} \text{ 不显含 } t)$$

$$\partial_t H = 0 \implies \boxed{E = \text{const}}$$

### 2. 空间平移对称性 → 动量守恒

$$x^i \to x^i + \delta x^i \quad (\mathcal{L} \text{ 不显含 } x^i)$$

$$\partial_t P^i = 0 \implies \boxed{\mathbf{P} = \text{const}}$$

### 3. 空间旋转对称性 → 角动量守恒

$$x^i \to x^i + \epsilon^{ijk}\,\theta^j\, x^k$$

$$\partial_t L^i = 0 \implies \boxed{\mathbf{L} = \text{const}}$$

### 4. $U(1)$ 相位对称性 → 电荷守恒

$$\phi \to e^{i\alpha}\phi \approx \phi + i\alpha\phi$$

$$\partial_\mu j^\mu = 0, \quad j^\mu = iq\left(\phi^*\partial^\mu\phi - \phi\,\partial^\mu\phi^*\right)$$

$$\boxed{Q = \text{const}}$$

### 5. 规范对称性 → 各种守恒荷

| 对称性 | 守恒量 |
|---|---|
| $U(1)_Q$ | 电荷 |
| $U(1)_Y$ | 弱超荷 |
| $SU(2)_L$ | 弱同位旋 |
| $SU(3)_C$ | 色荷 |

## 证明思路

### 经典力学版本

考虑拉格朗日量 $L(q, \dot{q}, t)$，做无穷小变换 $q \to q + \delta q$。

若 $\delta L = 0$（或 $\delta L = \frac{dF}{dt}$），则：

$$\delta L = \frac{\partial L}{\partial q}\delta q + \frac{\partial L}{\partial \dot{q}}\delta\dot{q} = 0$$

利用 Euler-Lagrange 方程 $\frac{\partial L}{\partial q} = \frac{d}{dt}\frac{\partial L}{\partial \dot{q}}$：

$$\frac{d}{dt}\left(\frac{\partial L}{\partial \dot{q}}\delta q\right) = 0$$

因此诺特荷：

$$\boxed{Q = \frac{\partial L}{\partial \dot{q}}\,\delta q = \text{const}}$$

### 场论版本

类似地，利用 Euler-Lagrange 场方程：

$$\partial_\mu\left(\frac{\partial \mathcal{L}}{\partial(\partial_\mu \phi)}\delta\phi - j^\mu_{\text{surface}}\right) = 0$$

## 离散对称性呢？

诺特定理**只适用于连续对称性**（可以用无穷小参数描述的变换）。

对于离散对称性（如宇称 $P$、时间反演 $T$、电荷共轭 $C$）：

- 诺特定理**不直接适用**
- 不给出对应的守恒流
- 但离散对称性仍有重要的物理后果（如选择定则）

---

## 对称性自发破缺与诺特定理

当对称性**自发破缺**（拉氏量有对称性，但基态没有）时：

| 情形 | 后果 |
|---|---|
| 全局连续对称性自发破缺 | **Goldstone 定理**：每个破缺的生成元对应一个无质量的 Goldstone 玻色子 |
| 规范对称性自发破缺 | **Higgs 机制**：Goldstone 玻色子被规范场"吃掉"，规范玻色子获得质量 |

例如：
- 铁磁体：旋转对称性自发破缺 → 自旋波（磁振子）
- QCD 手征对称性破缺 → $\pi$ 介子（近似 Goldstone 玻色子）
- 电弱对称性破缺 $SU(2)_L \times U(1)_Y \to U(1)_Q$ → $W^\pm, Z^0$ 获得质量

## 更深层的视角

诺特定理揭示了一个深刻的哲学命题：

> **守恒律不是偶然的巧合，而是自然界对称性的必然结果。**

| 对称性 | 守恒量 | 适用范围 |
|---|---|---|
| 时间平移 | 能量 | 所有物理 |
| 空间平移 | 动量 | 所有物理 |
| 空间旋转 | 角动量 | 所有物理 |
| $U(1)$ 相位 | 电荷 | 电磁相互作用 |
| $SU(3)$ 色荷 | 色荷 | 强相互作用 |
| CPT | CPT 对称 | 所有洛伦兹不变的局域场论 |

标准模型的全部结构——规范玻色子的种类、相互作用的耦合形式、守恒律——本质上都可以追溯到**对称性 + 诺特定理**。

$$\boxed{\text{对称性} \xleftrightarrow{\text{Noether}} \text{守恒律}}$$
