## 一、什么是"内禀属性"？

> **内禀属性**（intrinsic properties）是指粒子**固有的、不随参考系或运动状态改变**的物理量。

与之相对的是**外禀属性**（extrinsic/kinematic properties），它们依赖于粒子的运动状态或观测者的参考系：

|       内禀属性（固有）        |                   外禀属性（依赖运动状态）                    |
| :-------------------: | :-----------------------------------------------: |
|      静止质量 $m_0$       |              能量 $E = \gamma m_0 c^2$              |
|        自旋 $s$         |                  动量 $\mathbf{p}$                  |
|        电荷 $q$         |                  速度 $\mathbf{v}$                  |
| 磁矩 $\boldsymbol{\mu}$ | 角动量 $\mathbf{J} = \mathbf{L} + \mathbf{S}$ 中的轨道部分 |
|        宇称 $P$         |                波长 $\lambda = h/p$                 |

**换参考系不变的 → 内禀；换参考系会变的 → 外禀。**

## 二、内禀属性全览

### 1. 静止质量 $m_0$

- 粒子在**自身静止参考系**中的质量
- Lorentz 不变量：所有惯性系中观测到的静止质量相同
- 决定了粒子的动力学行为和时空轨迹（世界线）
- 光子 $m_0 = 0$，电子 $m_0 = 0.511\ \text{MeV}$

### 2. 电荷 $q$

- 粒子携带的电磁相互作用"荷"
- 量子化的：$q = ne$（$n$ 为整数，$e$ 为基本电荷）
- 守恒量：在所有已知相互作用中严格守恒
- 决定粒子在电磁场中的行为（Lorentz 力）

### 3. 自旋 $s$（固有角动量）

- 粒子携带的**内禀角动量**，与空间运动无关
- 量子化的：$s = 0, \frac{1}{2}, 1, \frac{3}{2}, 2, \ldots$（以 $\hbar$ 为单位）
- 自旋投影 $m_s$ 取 $2s+1$ 个值：$m_s = -s, -s+1, \ldots, +s$
- 决定粒子的统计性质：

$$\boxed{\text{Bose-Einstein 统计} \leftarrow s = 0, 1, 2, \ldots \quad (\text{整数自旋：玻色子})}$$

$$\boxed{\text{Fermi-Dirac 统计} \leftarrow s = \frac{1}{2}, \frac{3}{2}, \ldots \quad (\text{半整数自旋：费米子})}$$

- 自旋-统计定理是量子场论的基本定理之一

### 4. 磁矩 $\boldsymbol{\mu}$

- 与自旋相关联的内禀磁偶极矩
- 对自旋 $1/2$ 的粒子：

$$\boldsymbol{\mu} = g \frac{q}{2m} \mathbf{S}$$

其中 $g$ 因子是内禀属性：
- 电子：$g_e \approx 2.002319$（QED 修正给出反常磁矩 $a_e = (g-2)/2$）
- 质子：$g_p \approx 5.586$（反映内部夸克结构）
- 中子：$g_n \approx -3.826$（虽无电荷，但有磁矩！证明有内部结构）

### 5. 空间宇称 $P$

- 描述粒子在**空间反演** $\mathbf{r} \to -\mathbf{r}$ 下的行为
- 每个粒子有确定的内禀宇称：

| 粒子 | 内禀宇称 |
|:---:|:---:|
| 标量粒子（$\pi$ 介子、Higgs） | $P = -1$ |
| 赝标量粒子 | $P = -1$ |
| 矢量粒子（$\rho, \omega, \gamma$） | $P = -1$ |
| 费米子 | 约定 $P = +1$ |
| 反费米子 | $P = -1$ |

- 强相互作用和电磁相互作用守恒宇称
- 弱相互作用**不守恒**宇称（1956 年李政道、杨振宁提出，1957 年吴健雄实验验证）

### 6. 电荷共轭 $C$

- 描述粒子变为其**反粒子**的操作
- 粒子-反粒子具有相反的内禀量子数（电荷、重子数、轻子数等）
- $C$ 宇称：光子 $C = -1$，中性 $\pi^0$ 介子 $C = +1$
- 强和电磁相互作用守恒 $C$；弱相互作用不守恒 $C$

### 7. 荷宇称（G 宇称）

- 对同位旋空间中性态的粒子定义：$G = C \cdot e^{i\pi I_2}$
- $\pi$ 介子：$G = -1$

## 三、加法性量子数

这些量子数在反应中**守恒**（加法性：反应前后总量不变）：

### 8. 重子数 $B$

- 夸克：$B = +1/3$；反夸克：$B = -1/3$
- 重子（三个夸克组成）：$B = +1$（如质子、中子）
- 反重子：$B = -1$
- 介子（夸克-反夸克）：$B = 0$
- **在所有已知相互作用中严格守恒**

### 9. 轻子数

**总轻子数 $L$**：
- 轻子：$L = +1$；反轻子：$L = -1$

**族轻子数**（在标准模型中分别守恒，中微子振荡中不严格守恒）：

| 族 | 轻子数 | 包含粒子 |
|:---:|:---:|:---|
| 第一代 | $L_e$ | $e^-, \nu_e$ |
| 第二代 | $L_\mu$ | $\mu^-, \nu_\mu$ |
| 第三代 | $L_\tau$ | $\tau^-, \nu_\tau$ |

### 10. 同位旋 $I, I_3$（强相互作用中的近似对称性）

- 强相互作用近似地不区分 $u$ 夸克和 $d$ 夸克（因为 $m_u \approx m_d \ll \Lambda_{\text{QCD}}$）
- 核子构成**同位旋二重态**：

$$|p\rangle = |I = \tfrac{1}{2},\; I_3 = +\tfrac{1}{2}\rangle, \qquad |n\rangle = |I = \tfrac{1}{2},\; I_3 = -\tfrac{1}{2}\rangle$$

- $\pi$ 介子构成**同位旋三重态**：

$$|\pi^+\rangle,\quad |\pi^0\rangle,\quad |\pi^-\rangle \qquad (I = 1,\; I_3 = +1, 0, -1)$$

- 电磁相互作用**破坏**同位旋对称性（$I_3$ 守恒但 $I$ 不守恒）

### 11. 味量子数（Flavor quantum numbers）

描述三代夸克和轻子的区别：

| 量子数 | 定义 | 非零的粒子 |
|:---:|:---:|:---|
| 奇异数 $S$ | $S = -(n_s - n_{\bar{s}})$ | $s$ 夸克：$S = -1$ |
| 粲数 $C$ | $C = n_c - n_{\bar{c}}$ | $c$ 夸克：$C = +1$ |
| 底数 $B'$ | $B' = -(n_b - n_{\bar{b}})$ | $b$ 夸克：$B' = -1$ |
| 顶数 $T$ | $T = n_t - n_{\bar{t}}$ | $t$ 夸克：$T = +1$ |

> 强和电磁相互作用守恒所有味量子数；**弱相互作用可以改变味**（如 $s \to u$ 导致奇异数不守恒）。

## 四、色荷（Color charge）

### 12. 色荷

- 夸克携带的**强相互作用荷**，有三种"颜色"：红（R）、绿（G）、蓝（B）
- 反夸克携带反色：$\bar{R}, \bar{G}, \bar{B}$
- 胶子携带**色-反色**组合（共 8 种）
- QCD 的规范群为 $SU(3)_c$

> **色禁闭**：所有可观测的强子必须是**色单态**（无色）——重子是 RGB 的组合，介子是色-反色的组合。

## 六、内禀属性的深层意义

### 对称性 ↔ 守恒律（[[诺特定理]]）

每一个内禀量子数都对应一种**对称性**和一个**守恒律**：

| 内禀量子数 | 对应的对称性 | 守恒律 |
|:---:|:---:|:---:|
| 电荷 $Q$ | $U(1)$ 电磁规范对称 | 电荷守恒 |
| 重子数 $B$ | $U(1)_B$ 全局对称 | 重子数守恒 |
| 轻子数 $L$ | $U(1)_L$ 全局对称 | 轻子数守恒 |
| 同位旋 $I$ | $SU(2)$ 味对称（近似） | 同位旋守恒（近似） |
| 色荷 | $SU(3)_c$ 规范对称 | 色荷守恒 |
| 宇称 $P$ | 空间反演对称 | 宇称守恒（强、电磁） |
| 电荷共轭 $C$ | 粒子-反粒子对称 | $C$ 守恒（强、电磁） |

> **内禀属性的深层本质是：它们是基本相互作用对称性的量子数标记。** 粒子之所以拥有这些属性，是因为它们生活在具有特定对称性的物理定律之下。
