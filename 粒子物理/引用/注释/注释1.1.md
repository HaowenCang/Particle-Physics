# 从 $E^2 = |\boldsymbol{p}|^2 c^2 + m^2 c^4$ 到 $p^\mu p_\mu = -m^2 c^2$

## 关键：度规约定

两个表达式的等价性**完全依赖于四维动量的定义和度规的约定**。我们采用：

$$\eta_{\mu\nu} = \mathrm{diag}(-1,\;+1,\;+1,\;+1)$$

四维动量定义为：

$$p^\mu = \left(\frac{E}{c},\;p^1,\;p^2,\;p^3\right) = \left(\frac{E}{c},\;\boldsymbol{p}\right)$$

## 展开计算

$$p^\mu p_\mu \;=\; \eta_{\mu\nu}\,p^\mu p^\nu \;=\; \sum_{\mu,\nu=0}^{3}\eta_{\mu\nu}\,p^\mu p^\nu$$

由于 $\eta_{\mu\nu}$ 是对角的，只有 $\mu = \nu$ 的项存活：

$$p^\mu p_\mu \;=\; \eta_{00}\,(p^0)^2 + \eta_{11}\,(p^1)^2 + \eta_{22}\,(p^2)^2 + \eta_{33}\,(p^3)^2$$

代入度规分量：

$$= (-1)\left(\frac{E}{c}\right)^2 + (+1)(p^1)^2 + (+1)(p^2)^2 + (+1)(p^3)^2$$

$$= -\frac{E^2}{c^2} + |\boldsymbol{p}|^2$$

因此：

$$p^\mu p_\mu = -\frac{E^2}{c^2} + |\boldsymbol{p}|^2$$

## 代入质壳条件

将 $E^2 = |\boldsymbol{p}|^2 c^2 + m^2 c^4$ 代入：

$$p^\mu p_\mu = -\frac{|\boldsymbol{p}|^2 c^2 + m^2 c^4}{c^2} + |\boldsymbol{p}|^2$$

$$= -|\boldsymbol{p}|^2 - m^2 c^2 + |\boldsymbol{p}|^2$$

$$= -m^2 c^2$$

$$\boxed{p^\mu p_\mu = -m^2 c^2}$$

## 反向推导

反过来，从 $p^\mu p_\mu = -m^2c^2$ 出发：

$$-\frac{E^2}{c^2} + |\boldsymbol{p}|^2 = -m^2c^2$$

$$-\frac{E^2}{c^2} = -m^2c^2 - |\boldsymbol{p}|^2$$

$$\frac{E^2}{c^2} = |\boldsymbol{p}|^2 + m^2c^2$$

$$\boxed{E^2 = |\boldsymbol{p}|^2 c^2 + m^2 c^4}$$

两个表达式是**完全等价的**，只是同一个物理条件的不同写法——前者是分量形式，后者是洛伦兹协变（张量）形式。

> **补充说明**：如果采用相反的度规约定 $\eta_{\mu\nu} = \mathrm{diag}(+1,-1,-1,-1)$（Weinberg, Peskin & Schroeder 等教材），则 $p^\mu p_\mu = +m^2c^2$，符号翻转但物理内容完全相同。两种约定在文献中都很常见，阅读时需注意。
